# Nyam 수익모델 사업성 검토서

> 작성일: 2026-03-17
> 목적: "유튜브형 광고 수익 분배 모델"의 Nyam 적용 가능성을 숫자 기반으로 냉정하게 평가

---

## 1. 핵심 질문

**"무료 사용자에게 광고를 보여주고, 그 수익을 크리에이터(버블 운영자/평가단)에게 분배하는 유튜브식 모델이 Nyam에서 현실적인가?"**

결론부터: **현재 단계에서는 비현실적이다. 단, 조건부로 변형 모델은 가능하다.**

---

## 2. 유튜브 모델 해부

### 2-1. 유튜브의 구조

| 항목 | 수치 |
|------|------|
| 2025 광고 매출 | $47.7B (약 62조원) |
| MAU | 25억+ |
| 크리에이터 분배 비율 | 광고 수익의 55% |
| 누적 크리에이터 지급액 | $100B+ |
| 평균 CPM (미국) | $7-15 |
| 평균 CPM (한국) | $3-8 |

### 2-2. 유튜브가 작동하는 이유

1. **압도적 트래픽**: 하루 10억 시간 시청 → 광고 노출 기회 무한
2. **긴 체류 시간**: 평균 세션 40분+ → 광고 삽입 지점 다수
3. **영상 = 광고 친화적 매체**: pre-roll, mid-roll, post-roll, 디스플레이 등 다양한 포맷
4. **양면 네트워크 효과**: 시청자 ↔ 크리에이터가 서로를 끌어당김
5. **광고주 다양성**: 수백만 광고주가 Google Ads 통해 자동 입찰

**핵심**: 유튜브의 수익 분배가 가능한 이유는 **규모(scale)**이다. 크리에이터 1인당 수천~수백만 조회가 발생하기 때문에 55% 분배가 의미 있는 금액이 된다.

## 3. Nyam에 유튜브 모델을 적용하면

### 3-1. 현실적 시나리오 시뮬레이션

**가정 (낙관적)**:
- Nyam MAU: 50,000명 (출시 12개월 후)
- DAU/MAU: 20% → DAU 10,000명
- 세션당 광고 노출: 3회
- 월 평균 세션: 15회/유저
- 한국 모바일 앱 eCPM: $5-10

**월간 광고 수익 계산**:

```
광고 노출 수 = DAU × 세션 × 광고/세션 × 30일
           = 10,000 × 1 × 3 × 30
           = 900,000 노출/월

월 광고 수익 = 900,000 ÷ 1,000 × $7 (eCPM)
            = $6,300/월 (약 820만원)

크리에이터 분배 (55%) = 약 450만원/월
```

### 3-2. 크리에이터당 수익

```
활성 크리에이터 100명 기준:
  → 1인당 월 45,000원 (약 $35)

활성 크리에이터 50명 기준:
  → 1인당 월 90,000원 (약 $70)

활성 크리에이터 500명 기준:
  → 1인당 월 9,000원 (약 $7)
```

**비교**: 유튜브 중간 크리에이터 월 수입 $500-5,000 / OnlyFans 중간값 $180/월

### 3-3. 냉정한 평가

| 문제 | 심각도 | 설명 |
|------|--------|------|
| **절대적 규모 부족** | 치명적 | MAU 5만으로는 의미 있는 광고 수익 자체가 불가 |
| **낮은 체류 시간** | 높음 | 맛집 기록 앱 세션 2-5분 vs 유튜브 40분. 광고 노출 기회 극히 적음 |
| **광고 비친화적 매체** | 높음 | 기록/리뷰 텍스트+사진 사이 광고 삽입 → UX 파괴 |
| **브랜드 훼손** | 치명적 | PRD 핵심 가치 "제로 광고 수익, 광고-추천 이해충돌 없음" 정면 위배 |
| **리텐션 하락 위험** | 높음 | 식음료 앱 D30 리텐션 3.7%. 광고 추가 시 더 하락 |
| **크리에이터 동기부여 실패** | 치명적 | 월 4.5만원으로는 양질의 콘텐츠 생산 동기 없음 |

## 4. 유사 서비스 성공/실패 사례

### 4-1. 성공 사례

#### Letterboxd (영화 기록 + 소셜)
- **모델**: 프리미엄 구독 ($19/yr Pro, Patron) + 디스플레이 광고 + 영화제/배급사 프로모션
- **MAU**: 수백만 (팬데믹 이후 2배 성장)
- **핵심 교훈**: 크리에이터에게 수익 분배하지 않음. 대신 사용자의 "기록 자체의 즐거움"으로 충분한 콘텐츠 생산. 광고 매출은 Playwire 도입 후 490% 증가했지만, 이는 이미 수백만 MAU 달성 후.
- **Nyam 시사점**: 기록 도구로서의 가치가 먼저. 수익화는 규모 확보 후.

#### Vivino (와인 기록 + 마켓플레이스)
- **모델**: 와인 판매 수수료 15% + 프리미엄 구독 ($4.17/월)
- **사용자**: 6,500만+ 다운로드
- **핵심 교훈**: 기록 → 구매 전환이라는 자연스러운 수익 흐름. 광고가 아닌 거래(transaction) 기반.
- **Nyam 시사점**: 리뷰/기록 데이터를 거래와 연결하면 광고보다 높은 ARPU 가능.

#### Untappd (맥주 체크인 + 소셜)
- **모델**: B2B SaaS (업소용 $899-1,199/yr) + 소비자 앱 광고 + 타겟 마케팅 솔루션
- **B2B 고객**: 20,000개 업소 (75개국)
- **핵심 교훈**: 소비자 앱은 데이터 수집 채널, 진짜 수익은 B2B. 크리에이터 수익 분배 없음.
- **Nyam 시사점**: 소비자 기록 데이터의 가치는 B2B에서 실현 가능.

### 4-2. 실패/고전 사례

#### Creator Economy 플랫폼 공통 문제 (Andrew Chen, a16z)

| 문제 | 설명 |
|------|------|
| **졸업 문제** | 성공한 크리에이터가 플랫폼을 떠나 자체 채널 구축 |
| **수익 집중** | 상위 1% 크리에이터가 전체 수익의 대부분 차지 (OnlyFans 중간값 $180/월) |
| **Bio Link 전쟁** | 크리에이터 플랫폼은 인스타/틱톡의 "link in bio" 한 자리를 놓고 제로섬 경쟁 |
| **알고리즘 의존** | 소셜 플랫폼 알고리즘 변경 시 크리에이터 트래픽 급변동 |

#### 한국 맛집 플랫폼 교훈

| 플랫폼 | 수익모델 | 결과 |
|--------|---------|------|
| 망고플레이트 | 광고 + 프리미엄 | 2023년 서비스 종료 |
| 다이닝코드 | 광고 + B2B | 독립 운영 지속, 소규모 |
| 캐치테이블 | 예약 수수료 | 흑자 전환 성공 |
| 식신 | 광고 + B2B 데이터 | 운영 중, 성장 정체 |

**교훈**: 한국 맛집 앱에서 순수 광고 모델은 대부분 실패. 거래(예약/배달) 연결이 유일하게 성공한 수익 모델.

## 5. Nyam의 구조적 한계

### 5-1. 유튜브 모델 적용이 근본적으로 어려운 이유

| 유튜브 | Nyam | 차이 |
|--------|------|------|
| 콘텐츠 소비 = 핵심 활동 | 콘텐츠 생산(기록) = 핵심 활동 | 광고는 소비자에게 보여야 하는데, Nyam은 생산자 비중이 높음 |
| 영상 사이 광고 자연스러움 | 기록 사이 광고 부자연스러움 | UX 파괴 없이 광고 삽입 지점 극히 제한적 |
| 수억 명이 무료로 시청 | 버블(폐쇄형) 구조 → 도달 범위 제한 | 광고 노출 대상 자체가 구조적으로 작음 |
| 크리에이터 콘텐츠 = 시간 투자 | 5초 기록이 핵심 가치 | 크리에이터의 차별화된 "콘텐츠" 정의가 불명확 |
| 글로벌 시장 | 한국 수도권 25-39세 | 광고 시장 규모 자체가 제한적 |

### 5-2. 버블의 딜레마

Nyam의 핵심 차별점인 "버블(폐쇄형 그룹)"이 광고 모델과 근본적으로 충돌:

- **폐쇄형 → 낮은 도달률**: 콘텐츠가 버블 안에서만 소비 → 광고 노출 기회 극소
- **신뢰 기반 → 광고 거부감**: "가짜 리뷰 없는 공간"에 광고가 들어오면 신뢰 훼손
- **소규모 그룹 → 타겟팅 비효율**: 5-30명 버블에서의 광고는 타겟팅 데이터 부족

## 6. 현실적 대안: Nyam에 맞는 수익 모델

### 6-1. 추천 모델: 하이브리드 3-Layer

유튜브 모델을 그대로 가져오는 대신, Nyam의 특성에 맞는 변형 모델을 제안한다.

```
Layer 1: 프리미엄 구독 (Nyam Pro)           — 안정적 기반 수익
Layer 2: 크리에이터 경제 (유료 버블 + 팁)    — 크리에이터 수익 창출
Layer 3: B2B 데이터 & 프로모션              — 성장 후 스케일 수익
```

### 6-2. Layer 1: Nyam Pro 구독 (Month 6~)

| 항목 | 무료 | Pro (₩4,900/월) |
|------|------|-----------------|
| 기록 | 무제한 | 무제한 |
| 버블 | 3개 | 무제한 |
| Taste DNA | 기본 | 고급 분석 + 추세 변화 |
| AI 추천 | 기본 | 고급 (상황별, 예산별) |
| 블로그 리뷰 | 월 3회 | 무제한 |
| 광고 | 있음 | 없음 |
| 데이터 내보내기 | X | O |

**수익 시뮬레이션**:
```
MAU 50,000 × 전환율 3% = 1,500명 유료
1,500 × ₩4,900 = ₩7,350,000/월 (약 735만원)

MAU 100,000 × 전환율 5% = 5,000명 유료
5,000 × ₩4,900 = ₩24,500,000/월 (약 2,450만원)
```

Letterboxd 참조: 니치 소셜 앱의 유료 전환율은 보통 2-5%.

### 6-3. Layer 2: 크리에이터 경제 (Month 12~)

유튜브처럼 "플랫폼이 크리에이터에게 돈을 줌"이 아니라, **"유저가 크리에이터에게 직접 돈을 줌"** 구조.

#### A. 유료 버블 (핵심)

```
크리에이터가 유료 열람 전용 버블 운영
  → 구독자가 월 ₩3,000-10,000 결제
  → Nyam 수수료 10-15%
  → 크리에이터 85-90% 수취

예시:
  "소믈리에 김○○의 와인 버블" — 월 ₩5,000
  구독자 200명 × ₩5,000 = ₩1,000,000/월
  크리에이터 수입: ₩850,000/월
  Nyam 수수료: ₩150,000/월
```

**유튜브와의 비교**:
| 항목 | 유튜브 | Nyam 유료 버블 |
|------|--------|--------------|
| 돈의 출처 | 광고주 | 구독자(유저) |
| 크리에이터 수입 | 조회수 비례 | 구독자 수 비례 |
| 수입 예측성 | 불안정 (알고리즘 의존) | 안정적 (구독 기반) |
| 유사 모델 | YouTube AdSense | Patreon, 유료 뉴스레터 |

#### B. 팁 / 슈퍼챗 (보조)

```
버블 내 기록에 "고마워요" 팁 (₩1,000-5,000)
  → Nyam 수수료 5%
```

#### C. 크리에이터 인증 뱃지

```
일정 조건 충족 시 "인증된 맛잘알" 뱃지 부여
  조건: Lv.30 + 기록 200개+ + 버블 구독자 50명+
  → 유료 버블 개설 자격 부여
  → 프로필 신뢰도 강화
```

### 6-4. Layer 3: B2B (Month 24~)

| 수익원 | 모델 | 월 예상 수익 |
|--------|------|-------------|
| 식당 트렌드 리포트 | 구독 ₩99,000/월 | 업소 100개 × ₩99,000 = ₩990만 |
| 스폰서 챌린지 | "이 와인 기록하면 보상" 캠페인 | 건당 ₩500만-2,000만 |
| 예약 연동 수수료 | 캐치테이블 등 연동 | 건당 ₩500-1,000 |

## 7. 광고를 넣되, 유튜브식 분배는 하지 않는 모델

무료 사용자에게 광고를 보여주되, 그 수익을 크리에이터에게 분배하지 않고 Nyam 운영비로 사용하는 것이 현실적이다.

### 7-1. 광고 삽입 가능 지점 (UX 훼손 최소화)

| 지점 | 형식 | 예상 eCPM |
|------|------|----------|
| 발견 탭 피드 내 | 네이티브 광고 (기록 카드와 동일 포맷) | $8-12 |
| 기록 저장 완료 후 | 인터스티셜 (3초 자동 닫힘) | $10-15 |
| Today's Pick 하단 | 배너 광고 | $3-5 |
| 지도 위 마커 | 스폰서 마커 (식당 광고) | $15-25 |

### 7-2. 광고 수익 시뮬레이션

```
MAU 50,000 기준:
  무료 사용자: 48,500명 (Pro 제외)
  일 광고 노출: 48,500 × 0.2 (DAU/MAU) × 3회 = 29,100 노출/일
  월 노출: 29,100 × 30 = 873,000
  월 수익: 873,000 ÷ 1,000 × $8 = $6,984 (약 900만원)

MAU 100,000 기준:
  월 수익: 약 1,800만원
```

### 7-3. 광고 수익의 용도

```
광고 수익 → Nyam 운영비 (서버, AI API, 인건비)
           → 크리에이터에게는 "유료 버블" 수익이 돌아감
           → Pro 구독 → Nyam 성장 투자

크리에이터의 수입 = 유료 버블 구독료 (유저 → 크리에이터)
Nyam의 수입 = 광고 + Pro 구독 + 버블 수수료 + B2B
```

## 8. 단계별 로드맵과 예상 수익

### Phase 1: 무료 성장 (0-6개월)

```
목표: MAU 10,000
수익: ₩0
전략: 모든 기능 무료, 사용자 확보 집중
핵심 지표: D7 리텐션 > 15%, 월 기록 수 > 5/유저
```

### Phase 2: 수익화 시작 (6-12개월)

```
목표: MAU 30,000-50,000
수익원:
  - Nyam Pro: ₩350만-735만/월
  - 네이티브 광고 (발견탭, 지도): ₩500만-900만/월
  합계: ₩850만-1,635만/월

크리에이터: 아직 수익 없음. "인증 맛잘알" 뱃지로 동기부여
```

### Phase 3: 크리에이터 경제 (12-18개월)

```
목표: MAU 50,000-100,000
수익원:
  - Nyam Pro: ₩735만-2,450만/월
  - 광고: ₩900만-1,800만/월
  - 유료 버블 수수료 (10%): ₩150만-500만/월
  합계: ₩1,785만-4,750만/월

크리에이터 수입 (유료 버블):
  상위 크리에이터 10명: 월 ₩50만-100만
```

## 10. 최종 결론

### 10-1. 유튜브 모델 그대로는 NO

- **규모 문제**: 유튜브는 MAU 25억에서 작동. Nyam MAU 5만에서 같은 구조는 크리에이터당 월 4.5만원 → 동기부여 불가
- **매체 문제**: 영상 ≠ 기록. 광고 삽입 지점과 체류 시간이 근본적으로 다름
- **정체성 충돌**: "제로 광고 = 신뢰"가 Nyam의 핵심 차별점. 광고 수익 분배는 이를 정면 위배

### 10-2. 변형 모델은 YES (조건부)

```
크리에이터 수익 = 유료 버블 구독 (유저 → 크리에이터, Patreon 모델)
Nyam 수익     = Pro 구독 + 절제된 광고 + 버블 수수료 + B2B

"유튜브처럼 플랫폼이 크리에이터에게 돈을 주는 게 아니라,
유저가 크리에이터에게 직접 돈을 주고, 플랫폼은 인프라 비용을 거둔다."
```

### 10-3. 선행 조건

1. **MAU 5만 이상** 확보 없이는 어떤 수익 모델도 무의미
2. **D30 리텐션 10% 이상** (식음료 앱 평균 3.7%의 3배) 달성 필수
3. **크리에이터 생태계**는 Phase 3(12개월 후)에 시작. 너무 이른 수익화는 독

### 10-4. 한 줄 요약

> **Nyam의 수익 모델은 유튜브가 아니라 "Letterboxd(구독) + Patreon(크리에이터) + Untappd(B2B)"의 조합이다. 광고는 운영비 보전 수단이지, 크리에이터 분배 재원이 아니다.**
