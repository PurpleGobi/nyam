```
src/
├── app/                    # 라우팅만 (6 pages)
├── domain/                 # 순수 타입 (7 entities + 6 repository interfaces)

├── shared/                 # cn.ts, routes, categories

```

## 2. 현재 상태

### 동작하는 것
- `pnpm build` 통과 (0 에러, 0 경고)
- `pnpm lint` 통과
- 5개 라우트 정적 생성: `/`, `/discover`, `/groups`, `/profile`, `/record`
- Supabase DB 스키마 + RLS + 트리거 원격 적용 완료
- 회원가입 시 `users` + `user_stats` + `taste_dna` 자동 생성 (트리거)
- 기록 저장 시 `completeness_score`, `rating_overall` 자동 산출 (트리거)

### 아직 없는 것
- **application hooks**: 아직 구현 안 됨. 컨테이너가 직접 데이터를 불러오지 않고 빈 상태만 표시
- **인증 플로우**: middleware는 있으나 로그인/회원가입 UI 없음
- **기록 작성 플로우**: 사진 촬영 → AI 인식 → 기록 저장 전체 미구현
- **Taste DNA 재계산 로직**: DB 트리거가 아닌 application 레벨 (기록 CRUD 시 비동기 재계산)
- **Experience Atlas XP 적립**: 기록 생성 시 3축 XP 갱신 로직
- **실시간 데이터 연결**: 컨테이너 ↔ repository hook 연결
- **.env 파일**: `NEXT_PUBLIC_SUPABASE_URL`, `NEXT_PUBLIC_SUPABASE_ANON_KEY` 필요
- **이미지 업로드**: Supabase Storage 버킷 미생성
