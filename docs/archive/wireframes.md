# Nyam 와이어프레임

> 버전: 1.0.0 | 작성일: 2026-03-16
> 핵심 원칙: **"기록은 10초 이내에 끝낸다."**

---

## 핵심 원칙

| 원칙 | 설명 |
|------|------|
| **10초 기록** | 사진 → AI 인식 → 1탭 확인 → 완료. 평가/코멘트는 선택 |
| **기록 → 즉시 보상** | 기록 완료 직후 Taste Graph 업데이트 + 근처 추천 노출 |
| **Bottom-up 발견** | 기록 → 작성자 → 그룹 순서의 자연스러운 깊어짐 |
| **비공개 기본** | 모든 기록은 기본 비공개. 공유는 명시적 선택 |

---

## 네비게이션 구조

```
┌───────────────────────────────────────────────┐
│  🏠       🔍      (➕)      👥       👤       │
│  홈      발견    기록하기    그룹    프로필      │
└───────────────────────────────────────────────┘
                    ↑ FAB (코랄 원형, 돌출)
```

- 4탭 + 중앙 FAB
- FAB은 어느 화면에서든 접근 가능
- FAB 탭 → 즉시 카메라/기록 플로우 진입

### 홈 상태별 분기

| 상태 | 표시 내용 |
|------|----------|
| 기록 0건 (신규) | 온보딩 카드: "첫 기록을 해보세요!" + FAB 하이라이트 |
| 기록 1-4건 | 근처 추천 (기본) + "기록할수록 추천이 정확해져요" 넛지 |
| 기록 5건+ | 근처 추천 (Taste Graph 기반) + 그룹 피드 |
| 그룹 없음 | 그룹 섹션 대신: "친구를 초대해보세요" CTA |
| 위치 권한 없음 | 근처 추천 대신: "위치를 허용하면 맞춤 추천을 받아요" |

## 2. 기록 플로우 (핵심 — 10초)

**원칙: 최소 경로는 사진 → 확인 → 완료 (3탭, 10초)**
평가, 코멘트, 일기, 공유는 모두 선택 사항.

### Step 1: 사진 (0초)

FAB 탭 즉시 진입.

```
┌─────────────────────────────────┐
│  ✕                              │ ← 닫기
│                                 │
│                                 │
│                                 │
│        [ 카메라 뷰파인더 ]       │
│                                 │
│                                 │
│                                 │
│                                 │
│                                 │
├─────────────────────────────────┤
│                                 │
│  [갤러리]     (◉)     [수동입력] │ ← 셔터 + 대안
│                                 │
└─────────────────────────────────┘
```

- 갤러리: 최근 사진에서 선택
- 수동 입력: 사진 없이 텍스트로 기록 (식당명/메뉴 검색)
- 셔터 탭 → Step 2로

### Step 2: AI 인식 (2-3초, 자동)

```
┌─────────────────────────────────┐
│  ←                              │
│                                 │
│  ┌─────────────────────────────┐│
│  │                             ││
│  │      [ 촬영한 사진 ]        ││
│  │                             ││
│  └─────────────────────────────┘│
│                                 │
│  ✨ AI가 분석 중...              │ ← 로딩 (sparkles 맥동)
│  ████████░░░░░░░░░░  45%       │
│                                 │
│         (1-2초 소요)            │
│                                 │
└─────────────────────────────────┘

        ↓ 자동 전환 (인식 완료 시)

┌─────────────────────────────────┐
│  ←                              │
│                                 │
│  ┌─────────────────────────────┐│
│  │      [ 촬영한 사진 ]        ││
│  └─────────────────────────────┘│
│                                 │
│  🍝 크림 파스타              ✓  │ ← AI 1순위 (선택됨)
│  🍕 마르게리타 피자             │ ← AI 2순위
│  ✏️ 직접 입력                   │ ← 폴백
│                                 │
│  📍 을지로 골목식당          ✓  │ ← 위치 기반 식당 매칭
│  📍 을지로 파스타하우스          │ ← 2순위
│  🔍 다른 식당 검색              │ ← 폴백
│                                 │
│  ┌─────────────────────────────┐│
│  │        ✓ 기록 완료          ││ ← PRIMARY CTA
│  └─────────────────────────────┘│
│                                 │
│  ┌─────────────────────────────┐│
│  │    + 평가/코멘트 추가 ▾     ││ ← 접힌 상태 (선택)
│  └─────────────────────────────┘│
└─────────────────────────────────┘
```

**핵심: "기록 완료" 버튼이 즉시 보인다.** 평가/코멘트 없이 바로 완료 가능.

### Step 2-b: 평가/코멘트 추가 (선택, 펼치면)

"+ 평가/코멘트 추가" 탭 시 아래로 확장:

```
│  ── 평가 (선택) ──────────────  │
│                                 │
│  맛      ○ ○ ○ ○ ●             │ ← 5점 탭 (별 또는 원)
│  가성비   ○ ○ ○ ● ○             │
│  서비스   ○ ○ ● ○ ○             │
│  분위기   ○ ○ ○ ○ ●             │
│                                 │
│  ── 한줄평 (선택) ────────────  │
│  ┌─────────────────────────────┐│
│  │ "파스타가 정말..."  ✨AI보조 ││ ← 텍스트 입력 + AI 서제스트
│  └─────────────────────────────┘│
│                                 │
│  ── 태그 (선택) ──────────────  │
│  [🥂데이트] [💼점심] [🍜혼밥]   │ ← 상황 태그 탭
│  [+직접입력]                    │
│                                 │
│  ── 공유 대상 ────────────────  │
│  ◉ 나만 보기                    │ ← 기본값
│  ○ 직장동료 그룹                │
│  ○ 대학친구 그룹                │
│  ○ 전체 공개                    │
│                                 │
│  ┌─────────────────────────────┐│
│  │        ✓ 기록 완료          ││
│  └─────────────────────────────┘│
```

### Step 2-c: 일기 추가 (선택, 더 아래)

평가 영역 아래에 접힌 "일기" 섹션:

```
│  ── 🔒 나만의 일기 (선택) ───  │ ← 탭하면 펼침
│                                 │
│  👥 함께한 사람                  │
│  [민지 ✕] [준호 ✕] [+추가]     │ ← 연락처/기존 사용자 검색
│                                 │
│  🎭 모임                        │
│  [대학동기] [회사점심] [+직접]  │ ← 이전에 사용한 태그 자동 제안
│                                 │
│  💭 분위기                       │
│  [따뜻한] [편안한] [시끌벅적]   │ ← 감정 태그 (탭으로 선택)
│  [특별한] [낭만적] [+직접]      │
│                                 │
│  📝 메모                         │
│  ┌─────────────────────────────┐│
│  │ "준호가 추천한 파스타가      ││
│  │  진짜 맛있었다. 다음엔..."  ││
│  └─────────────────────────────┘│
```

### Step 3: 완료 (즉시 보상)

"기록 완료" 탭 후:

```
┌─────────────────────────────────┐
│                                 │
│           ✓                     │ ← 체크 애니메이션 (300ms)

│                                 │

│  │  +15 XP  (기록자까지 45XP)  ││ ← 경험치 획득
│  └─────────────────────────────┘│
│                                 │
│  📍 이 근처 좋아한 곳            │
│  ┌────────┐ ┌────────┐         │ ← 즉시 추천 (가로 스크롤)
│  │ [사진] │ │ [사진] │         │
│  │ 을지로  │ │ 을지로  │         │
│  │ 스시바  │ │ 카페   │         │
│  │ ⭐4.8  │ │ ⭐4.0  │         │
│  └────────┘ └────────┘         │
│                                 │
│  ┌─────────────────────────────┐│
│  │        홈으로              ││
│  └─────────────────────────────┘│
└─────────────────────────────────┘
```

### 기록 플로우 타임라인

```
0초   FAB 탭 → 카메라 열림
2초   셔터 탭 (사진 촬영)
4초   AI 인식 완료 → 메뉴+식당 후보 표시
6초   1순위 맞으면 그대로, 아니면 1탭으로 변경
7초   "기록 완료" 탭
      ─── 10초 이내 최소 경로 완료 ───
      (선택) 평가 슬라이더 추가: +5초
      (선택) 한줄평 추가: +10초
      (선택) 일기 작성: +30초~
      (선택) 공유 대상 변경: +3초
```

## 3. 발견

공개된 기록을 탐색하고, 기록 → 작성자 → 그룹으로 이어지는 bottom-up 발견.

```
│                                 │

│                                 │
│  [📍근처] [⭐높은순] [🆕최신]  │ ← 정렬 옵션
│                                 │
│ ┌─────────────────────────────┐ │
│ │ [사진]                      │ │ ← 기록 카드
│ │ 크림파스타 · 을지로 골목식당 │ │

│                                 │
```

나의 Taste Graph, 통계, 레벨, 기록 타임라인.

```
┌─────────────────────────────────┐
│  내 프로필               ⚙️     │
├─────────────────────────────────┤
```

## 핵심 플로우 요약

```
기록 플로우 (10초):
  FAB → 📷 → AI 인식 → ✓ 확인 → 완료 → 보상

발견 플로우 (Bottom-up):
  기록 발견 → ❤️ 저장 → 👤 작성자 프로필
  → 🧩 궁합 확인 → 👥 그룹 발견 → 참여

활용 플로우 (즉시):
  홈 진입 → 📍 근처 추천 (내 기록 기반)
  → 탭 → 식당 상세 → 기록하기

그룹 플로우:
  그룹 생성 → 멤버 초대 → 기록 공유
  → 그룹 통계/랭킹 → 활동 촉진
```
