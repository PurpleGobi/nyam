## 1. 디자인 원칙

| 원칙 | 설명 |
|------|------|
| 기록의 즐거움 | 사진은 크고 선명하게, AI 인식은 즉각적으로, 마찰 최소화 |
| 즉각적 보상 | 기록 직후 Taste DNA 업데이트, XP 획득, 레벨 변화 |
| 따뜻한 기술 | 음식의 온기 + AI의 편안함. 차갑지 않은 톤 |
| 명확한 정보 위계 | 기록 → 사람 → 그룹 순서로 깊어지는 탐색 |
| 신뢰 연결 | 버블 멤버 자연 노출, Bottom-up 발견 흐름 |

## 4. 레이아웃

### 4-1. 전체 구조

```
┌─────────────────────────────────────────┐
│           공통 헤더 (sticky)              │
│  [nyam 로고]              [검색] [알림]   │
├─────────────────────────────────────────┤
│                                         │
│         <main> max-w-lg (512px)         │
│              mx-auto                     │
│              pb-20                       │
│                                         │
├─────────────────────────────────────────┤
│          하단 네비게이션 (fixed)           │
│  [홈] [발견] [+기록] [버블] [프로필]       │
└─────────────────────────────────────────┘
```

### 4-2. 헤더 (AppShell)

- `sticky top-0 z-50`
- 배경: `bg-background/95 backdrop-blur`
- 내부: `max-w-lg mx-auto h-14 px-5`
- 좌: `LogoLink` (nyam, Comfortaa, text-orange-500)
- 우: `HeaderActions` (검색 + 알림 아이콘, 미읽은 알림 도트)
- 모든 페이지 동일 (홈 포함)

### 4-3. 하단 네비게이션 (BottomNav)

- `fixed bottom-0 z-50 rounded-t-2xl`
- 내부: `max-w-lg mx-auto`
- 5개 아이콘: 홈, 발견, **기록(FAB)**, 버블, 프로필
- FAB: `-mt-5 h-14 w-14 rounded-full bg-[#FF6038]`, 흰색 Plus 아이콘
- 활성 탭: `text-[#FF6038]`, 비활성: `text-neutral-400`

### 4-4. 페이지 간격

- 모든 컨테이너: `pt-6` (헤더와 첫 콘텐츠 사이)
- 섹션 간: `gap-3` ~ `gap-6` (페이지별 상이)
- 좌우 패딩: `px-4`

## 6. 컴포넌트 상세

### 6-1. RatingScales (100점 슬라이더)

- 유형별 항목 자동 전환 (식당 6항목, 와인 6항목, 요리 6항목)
- 각 항목: `label (w-14)` + `track (flex-1 h-3)` + `thumb (w-6 h-6)` + `value (w-8)`
- 터치 드래그 지원 (`touch-none`, `onTouchStart/Move/End`)
- 활성: `bg-[#FF6038]`, 비활성: `bg-neutral-100`
- 기록 생성과 수정에서 동일 컴포넌트 사용

### 6-2. PhotoCaptureSheet (사진 선택)

- `<label>` + hidden `<input type="file">` 패턴 (ref.click() 사용 금지)
- 카메라: `capture="environment"`, 앨범: `multiple`
- 선택 후: 3열 그리드, X 버튼으로 삭제
- `e.target.value = ''`로 onChange 재트리거 보장

### 6-3. PhotoCarousel (사진 캐러셀)

- 가로 스크롤: `flex gap-2 overflow-x-auto`
- 1장: 100% 너비, 2장+: 85% 너비
- `h-64 rounded-2xl object-cover`
- Next.js Image 컴포넌트 (`fill` 모드)

### 6-4. HomeProfileCard

- 상단: 아바타(이니셜) + 닉네임 + 통계(기록/장소/버블) + 레벨/XP
- 하단: Taste DNA 탭 (Food/Wine) + 레이더 차트 (SVG)
- 기록 부족 시: "기록이 쌓이면 맛 취향이 분석돼요" 메시지

### 6-5. TodaysPickCard

- 배경: 음식 사진 + 그라데이션 오버레이
- 정보: 추천 이유 + 식당/메뉴 이름
- 인터랙션: 탭하면 다음 추천으로 순환
- 하단: "탭하면 다른 추천 · 총 N곳" + 자세히 보기 버튼

### 6-6. RatingBar (상세 페이지용)

- 읽기 전용 수평 바
- `label (w-14)` + `bar (h-2 rounded-full)` + `value (w-6)`
- 채움: `bg-[#FF6038]`, 배경: `bg-neutral-100`

## 9. 반응형 & 접근성

### 9-1. 레이아웃 제한

- 모든 콘텐츠: `max-w-lg` (512px) + `mx-auto`
