와인 기능의 목표는 "기록과 성장": 기록은 편하게 → 자동 생성된 통계를 보면서 자동 공부.

**Why:** 기존 와인 앱(Vivino 등) 리서치 결과 별다른 혁신 없음. Nyam만의 차별점은 식당+와인 맥락 연결. 사용자가 "공부한다"는 느낌 없이 기록이 쌓이면서 자연스럽게 와인 지식이 늘어나는 구조가 핵심.

**How to apply:**
- 기록 UX는 최소 마찰 (사진 스캔 중심, 최소 입력)
- 통계/인사이트는 자동 생성 (산지 지도, 품종 분포, 점수 분포 등)
- 통계 자체가 학습 도구 역할 (내가 어떤 와인을 좋아하는지, 어떤 산지를 아직 탐험 안 했는지)
- 블렌드 품종 비율 등 전문적 정보는 전면에 내세우지 않음 — 기록하면 자동으로 쌓임
- 다른 와인 앱 기능 무리하게 따라가지 않음
