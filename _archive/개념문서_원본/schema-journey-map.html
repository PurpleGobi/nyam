25개 테이블 스키마 요약 (주요 컬럼만)

Core Data

users

유저 프로필, 설정, XP, 프라이버시
  • id PKUUID
  • nickname, handleVARCHAR
  • avatar_url, avatar_colorTEXT
  • taste_summary, taste_tagsTEXT, TEXT[]
  • privacy_profilepublic|bubble_only|private
  • privacy_recordsall|shared_only
  • visibility_public, _bubbleJSONB (토글맵)
  • pref_* (8개)VARCHAR (설정값)
  • notify_* (4개)BOOLEAN
  • total_xp, active_xpINT
  • record_count, follower_countINT (비정규화)
  • auth_provider, auth_provider_idVARCHAR

restaurants

식당 마스터 데이터
  • id PKUUID
  • nameVARCHAR(100)
  • country, city, area, districtVARCHAR (지역)
  • genre16종 CHECK
  • price_range1~4 (만원 단위)
  • lat, lngDOUBLE (PostGIS)
  • photos, menus, hoursTEXT[], JSONB
  • michelin_stars, has_blue_ribbonINT, BOOL
  • nyam_scoreDECIMAL(4,1)
  • external_idsJSONB (카카오/네이버/구글)

wines

와인 마스터 데이터
  • id PKUUID
  • name, producerVARCHAR
  • region, sub_region, countryVARCHAR
  • wine_typered|white|rose|sparkling|...
  • variety, grape_varietiesVARCHAR, JSONB
  • vintage, abvINT, DECIMAL
  • body/acidity/sweetness_levelINT (1~5 / 1~3)
  • food_pairingsTEXT[]
  • vivino_rating, critic_scoresDECIMAL, JSONB
  • nyam_scoreDECIMAL(4,1)

records

기록 (식당 방문 + 와인 테이스팅)
  • id PKUUID
  • user_id FK→ users
  • target_id, target_typeUUID, restaurant|wine
  • statuschecked|rated|draft
  • axis_x, axis_yDECIMAL (사분면)
  • satisfactionINT 1~100 (점 크기)
  • scenesolo|romantic|friends|...
  • aroma_regions, aroma_labelsJSONB, TEXT[] (와인)
  • complexity, finish, balanceINT/DECIMAL (와인)
  • comment, menu_tags, tipsVARCHAR, TEXT[]
  • total_price / purchase_priceINT (KRW)
  • visit_date, meal_timeDATE, VARCHAR
  • is_exif_verifiedBOOL (GPS 검증)

record_photos

기록 사진
  • id PKUUID
  • record_id FK→ records (CASCADE)
  • urlTEXT
  • order_indexINT

grape_variety_profiles

포도 품종 레퍼런스 (20종)
  • name PKVARCHAR
  • name_koVARCHAR (한국어)
  • categoryred|white
  • typical_body/acidity/tanninINT
Social & Bubbles

bubbles

버블 (신뢰 기반 그룹)
  • id PKUUID
  • name, descriptionVARCHAR
  • visibilityprivate|public
  • join_policyinvite_only|closed|manual|auto|open
  • min_records, min_levelINT (가입 조건)
  • member_count, record_countINT (비정규화)
  • invite_codeVARCHAR UNIQUE
  • created_by FK→ users

bubble_members

버블 멤버십
  • (bubble_id, user_id) PK복합키
  • roleowner|admin|member|follower
  • statuspending|active|rejected
  • taste_match_pctDECIMAL (크론 계산)
  • badge_labelVARCHAR (뱃지)

bubble_shares

기록 → 버블 공유
  • id PKUUID
  • record_id FK→ records
  • bubble_id FK→ bubbles
  • shared_by FK→ users
  • UNIQUE(record_id, bubble_id)중복 방지

follows

팔로우 관계
  • (follower_id, following_id) PK복합키
  • statuspending|accepted|rejected

comments

버블 내 댓글
  • id PKUUID
  • target_id, bubble_id FK→ records, bubbles
  • user_id FK→ users
  • contentVARCHAR(300)

reactions

리액션 (기록/댓글)
  • id PKUUID
  • target_typerecord|comment
  • reaction_typelike|bookmark|want|check|fire
  • UNIQUE(target, reaction, user)중복 방지

bubble_share_reads

피드 읽음 추적
  • (share_id, user_id) PK복합키
  • read_atTIMESTAMPTZ

bubble_ranking_snapshots

주간 랭킹 캐시
  • (bubble, target, type, period) PK복합키
  • rank_positionINT
  • avg_satisfactionDECIMAL
XP & Gamification

user_experiences

축별 XP + 레벨
  • id PKUUID
  • user_id FK→ users
  • axis_typecategory|area|genre|wine_variety|wine_region
  • axis_valueVARCHAR (한식, 을지로, ...)
  • total_xp, levelINT

xp_histories

XP 지급 로그
  • id PKUUID
  • user_id FK→ users
  • record_id FK→ records (nullable)
  • reason15종 CHECK
  • xp_amountINT

level_thresholds

레벨 테이블 (L1~L99)
  • level PKINT (1~99)
  • required_xpINT (지수 곡선)
  • title입문자 → 식도락 마스터

milestones + user_milestones

업적 정의 + 달성 기록
  • milestones.id PK업적 정의 (축, 임계값, XP)
  • user_milestones (user, milestone, axis) PK달성 기록
Discovery & Notifications

wishlists

찜 목록
  • id PKUUID
  • user_id FK→ users
  • target_id, target_typerestaurant|wine
  • sourcedirect|bubble|ai|web
  • is_visitedBOOL

saved_filters

저장된 필터 프리셋
  • id PKUUID
  • target_typerestaurant|wine|bubble|...
  • rulesJSONB (필터 규칙)

ai_recommendations

AI 추천
  • id PKUUID
  • target_id, target_typerestaurant|wine
  • reason, algorithmTEXT, VARCHAR
  • confidenceDECIMAL (0~1)

notifications

알림
  • id PKUUID
  • user_id FK→ users (수신자)
  • notification_type11종
  • actor_id FK→ users (발생자)
  • is_read, action_statusBOOL, pending|accepted|rejected

nudge_history + nudge_fatigue

넛지 로그 + 피로도
  • nudge_history.id PK넛지 발송 기록
  • nudge_typelocation|time|photo|unrated|...
  • nudge_fatigue.user_id PK피로도 점수 + 일시정지

Trigger 비정규화

records INSERT/DELETE → users.record_count ±1
follows INSERT/UPDATE/DELETE → users.follower_count, following_count ±1
bubble_members INSERT/UPDATE/DELETE → bubbles.member_count, follower_count ±1
bubble_shares INSERT/DELETE → bubbles.record_count ±1, last_activity_at
CRON: refresh_active_xp() → users.active_xp, active_verified (6개월 기준)
