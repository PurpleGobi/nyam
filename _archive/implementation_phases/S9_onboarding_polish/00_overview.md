## 온보딩 스텝 (ONBOARDING.md 기준)

```
Step 0: 로그인 (구글/카카오/네이버/애플)
Step 1: 인트로 ("낯선 별점 천 개보다, 믿을만한 한 명의 기록")
Step 2: 맛집 등록 (지역 선택 → 해당 지역 식당 체크 → 초기 데이터)
Step 3: 버블 생성 (가족/친구/직장동료 템플릿 + 초대)
Step 4: 버블 탐색 (공개 버블 둘러보기)
        → 홈 이동
```

## 완료 기준

```
□ 신규 유저: 로그인→인트로→맛집 등록→버블 생성→탐색→홈 완주
□ 온보딩 30초 내 완료 가능
□ 넛지: 미평가/식사시간/사진 감지 조건별 표시
□ E2E: 기록→공유→프로필→버블→팔로우 전체 시나리오
□ pnpm build 성공
□ LCP < 3s (3G), < 1s (WiFi)
□ 번들 사이즈 < 500KB initial
□ 콘솔 에러/경고 0개
```
