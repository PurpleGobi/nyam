## 1. 전체 화면 맵

```
App
├── 온보딩 (/onboarding)
│   ├── 가입 (소셜 로그인)
│   ├── 동네 선택
│   └── 가본 곳 체크
│
├── 메인 네비게이션 (헤더 아바타 + FAB + 콘텐츠 탭)
│   ├── [홈] (/)
│   │   ├── 앱 헤더 (nyam 로고 · 알림 · 아바타 → 프로필)
│   │   ├── AI 인사 (시간대별 멘트, 5초 소멸)
│   │   ├── 넛지 스트립 (사진 감지/미평가/식사 후 — 1개)
│   │   ├── 콘텐츠 탭 [식당 | 와인]
│   │   │   ├── 식당 탭
│   │   │   │   ├── 서브 네비: 방문 | 찜 | 추천
│   │   │   │   ├── 상황 필터 (혼밥/데이트/회식 등)
│   │   │   │   └── 플레이스 카드 리스트 (3소스)
│   │   │   └── 와인 탭
│   │   │       ├── 서브 네비: 시음 | 찜 | 셀러
│   │   │       ├── 카메라 3모드 (개별/진열장/영수증)
│   │   │       ├── 최근 등록 와인 카드
│   │   │       ├── 산지 지도 드릴다운
│   │   │       ├── 품종 차트
│   │   │       ├── 점수 분포
│   │   │       └── 월별 소비
│   │   └── FAB (+) → 추가 바텀시트 (식당: 위치기반+검색 / 와인: 라벨촬영+검색)
│   │
│   ├── [내 컬렉션] (/collection)
│   │   ├── 탭 [식당 | 와인]
│   │   ├── 노션 스타일 필터 (속성+조건+And/Or)
│   │   ├── 소팅 (최신/점수/이름/방문순)
│   │   └── 컴팩트 리스트
│   │
│   ├── [탐색] (/discover)
│   │   ├── 지역별 식당/와인
│   │   ├── 권위 추천 (미슐랭/블루리본)
│   │   └── 취향 유사 추천 (P2+)
│   │
│   ├── [+ 추가] (바텀시트)
│   │   ├── 식당 추가
│   │   │   ├── 근처 식당 목록 (GPS, primary) → 선택 → 저장
│   │   │   └── 이름 검색 (폴백) → 선택 → 저장
│   │   └── 와인 추가
│   │       ├── 라벨 촬영 (카메라, primary) → AI 매칭 → 확인 → 저장
│   │       └── 이름 검색 (폴백) → 선택 → 저장
│   │
│   ├── [버블] (/bubbles) — P2
│   │   ├── 버블 목록
│   │   ├── 버블 상세 (/bubbles/[id])
│   │   │   ├── 피드 탭
│   │   │   ├── 식당 목록 탭
│   │   │   ├── 와인 목록 탭
│   │   │   └── 통계 탭
│   │   └── 버블 생성
│   │
│   └── [나] (/profile)
│       ├── 미식 정체성 + Wrapped 공유
│       ├── 경험치 & 레벨
│       ├── 내 버블
│       └── 설정
│
├── 상세 페이지
│   ├── 식당 상세 (/restaurants/[id])
│   └── 와인 상세 (/wines/[id])
│
├── 기록 상세 (/records/[id])
│
└── 알림 (/notifications)
```

## 2. 핵심 사용자 플로우

### Flow A: 추가하기 (Phase 1 — 핵심 루프)
```
+ FAB → 식당/와인 선택 → 대상 특정 (위치기반/라벨촬영/검색) → 저장 (끝)
```

### Flow B: 풍성화 (Phase 2 — 선택적, 언제든)
```
상세페이지 / 목록 / 넛지 → "내용 추가하기" → 사분면 평가 → 상황 태그 → (선택: 한줄평/사진/메뉴) → 저장
```

### Flow C: 재방문/재음용 기록
```
식당 상세 → "다시 기록하기" → 사분면 (이전 위치 반투명 표시) → 상황 태그 → 저장
```

### Flow D: 추천 → 기록
```
홈 식당 탭 → 추천 서브탭 or 상황 필터 → 식당 선택 → 상세 → 기록 or 찜
```

### Flow E: 사진 넛지 → 기록
```
앱 진입 → 갤러리 사진 감지 카드 → "기록하기" → EXIF+AI 매칭 → 저장 (→ 풍성화 유도)
```

### Flow F: 이미 있는 항목 추가 시도
```
+ FAB → 식당/와인 특정 → 이미 기록 있음 → "다녀온 적 있어요" 토스트 → 상세페이지 이동
```

## 4. 구현 순서 (스프린트 계획 — P1+P2 통합)

> P2 테이블은 S1에서 전부 생성. P2 기능은 S7~S8에서 구현.
> 온보딩은 모든 기능 완성 후 마지막에 연결.

```
S1 ─── S2 ─── S3 ─┬─ S4
 (기반)  (코어)  (검색) │ (상세P1)
                    │
                    S5 ─── S6 ─── S7 ─── S8 ─── S9
                   (홈+추천) (XP+프로필) (버블) (통합) (온보딩+마무리)
```

| Sprint | 문서 | 산출물 | 선행 | 비고 |
|--------|------|--------|------|------|
| **S1** | DATA_MODEL + AUTH + DESIGN_SYSTEM | **P1+P2 전체** DB 스키마 (버블·소셜 테이블 포함), RLS, 인증, 디자인 토큰 | 없음 | P2 테이블도 이때 생성 → 나중에 마이그레이션 고통 방지 |
| **S2** | RATING_ENGINE + RECORD_FLOW | 사분면 UI, 아로마 팔레트, 제스처, 기록 저장 플로우 | S1 | 핵심 UX — 여기서 사분면 완성도가 앱 품질 결정 |
| **S3** | SEARCH_REGISTER | 검색 (Nyam DB → 외부 API), 식당/와인 등록, 사진 등록 (EXIF+AI) | S1, S2 | 검색→선택→S2 평가로 연결되는 풀 플로우 완성 |
| **S4** | RESTAURANT_DETAIL + WINE_DETAIL | 상세 페이지 L1~L8 (히어로, 평점, 내 기록, 사분면 분포, 실용 정보, 와인 링크) | S1, S2 | P2 소셜 레이어(L9)는 S8에서 추가 |
| **S5** | HOME + RECOMMENDATION | 홈 식당/와인 탭 구조, AI 인사, 넛지 스트립, 추천 알고리즘 7종 (재방문/상황/사분면/위시/권위/와인/지오펜스), Discover | S1~S4 | 추천은 기록 수 기반 점진 해금 |
| **S6** | XP_SYSTEM + PROFILE + SETTINGS | XP 계산, 레벨, 미식 정체성, Wrapped 공유, 프로필 화면, 설정 전체 | S2 | |
| **S7** | BUBBLE | 버블 생성/가입/초대, 공유 플로우, 피드(크로노), 댓글/리액션, 버블 통계, 알림, 역할/권한 | S1, S6 | P2 코어 — 버블 시스템 전체를 한 스프린트에 |
| **S8** | BUBBLE 통합 + FOLLOW | 상세 페이지 L9 (버블 멤버 기록), 홈 버블 피드, 기록→버블 공유, 팔로우, 공개 프로필 | S4, S5, S7 | 기존 P1 페이지에 P2 레이어 통합 |
| **S9** | ONBOARDING + 마무리 | 온보딩 풀 플로우 (가입→동네→식당→와인→홈), 넛지 시스템, 전체 플로우 검증, 빌드 | 전체 | 모든 기능 완성 후 온보딩 연결 |

## 5. 홈 네비게이션 구성

> 홈 화면에는 하단 탭 바가 없다. 대신 FAB + 콘텐츠 탭 + 헤더 아바타로 네비게이션을 구성한다.

| 요소 | 위치 | 역할 |
|------|------|------|
| 앱 헤더 (nyam 로고 · 🔔 · 아바타) | 상단 sticky | 아바타 탭 → 프로필/설정 진입 |
| 콘텐츠 탭 [식당 \| 와인] | 헤더 아래 | 식당/와인 도메인 전환, 우측 아이콘 (식당: 필터, 와인: 목록) |
| FAB (+) | 우하단 고정 | 추가 바텀시트 → 식당(위치기반+검색) / 와인(라벨촬영+검색) |

### 전체 앱 페이지 네비게이션

| 페이지 | 진입 경로 |
|--------|-----------|
| 홈 | 기본 랜딩 (/) |
| 탐색 (Discover) | 홈에서 추천 카드 또는 별도 진입 |
| 추가하기 | FAB (+) → 추가 바텀시트 (식당/와인 선택 → 특정 → 저장) |
| 내 컬렉션 | 홈 와인 목록 아이콘, 프로필 "내 기록 전체", 와인 대시보드 수치 탭 |
| 버블 | /bubbles (P2) |
| 프로필/설정 | 헤더 아바타 탭 |
