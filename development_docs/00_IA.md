# Nyam v2 Information Architecture

> 화면 맵, 네비게이션 구조, 사용자 플로우, 화면 간 의존성
> **SSOT**: `prototype/*.html` 목업 기반, `pages/*.md` 문서와 동기화

---

## 1. 전체 화면 맵

```
App
├── 온보딩 (/onboarding)
│   ├── 로그인 (카카오 / 구글 / 애플)
│   ├── 인트로 ("낯선 별점 천 개보다, 믿을만한 한 명의 기록")
│   ├── 경험치 설명 (기록 → XP → 레벨 소개)
│   ├── 버블 생성 (가족/친구/직장동료 템플릿 + 초대)
│   └── 버블 탐색 (공개 버블 둘러보기 → 홈 이동)
│
├── 홈 (/) — 앱 메인 허브
│   ├── 앱 헤더 (nyam 로고 · bubbles · 🔔알림 · 아바타 드롭다운)
│   │   ├── bubbles 텍스트 버튼 → /bubbles
│   │   ├── 🔔 → 알림 드롭다운 (인라인, 별도 페이지 아님)
│   │   └── 아바타 [J] → 드롭다운 (프로필, 설정)
│   │
│   ├── 콘텐츠 탭 [식당 | 와인]
│   │   ├── 뷰 순환 버튼 (상세 → 리스트 → 캘린더)
│   │   ├── 지도 뷰 버튼
│   │   ├── 통계 패널 토글
│   │   ├── 노션 스타일 필터 (다중 조건 AND/OR)
│   │   └── 소팅 드롭다운
│   │
│   ├── 식당 탭
│   │   ├── 서브 네비: 방문 | 찜 | 추천 | 팔로잉
│   │   ├── 상세 뷰 (카드형 — 사진, 점수, 씬태그, 사분면, 메뉴)
│   │   ├── 리스트 뷰 (컴팩트 — 이름, 점수, 장르, 방문수)
│   │   ├── 캘린더 뷰 (월별 — 날짜별 사진 도트)
│   │   ├── 지도 뷰 (방문 식당 핀)
│   │   └── 통계 패널 (지도, 장르 차트, 점수 분포, 월별 트렌드)
│   │
│   ├── 와인 탭
│   │   ├── 서브 네비: 시음 | 찜 | 셀러
│   │   ├── 상세/리스트/캘린더 뷰 (식당과 동일 구조)
│   │   └── 통계 패널 (산지 지도 3-level 드릴다운, 품종 차트, 점수 분포)
│   │
│   └── FAB (+) → 기록 추가 바텀시트
│       ├── 식당: 근처 식당(GPS) + 검색
│       └── 와인: 라벨 촬영 + 검색
│
├── 식당 상세 (/restaurants/[id])
│   ├── L1: 히어로 (사진 캐러셀, 찜/공유)
│   ├── L2: 기본 정보 (이름, 장르·지역·가격대)
│   ├── L3: 점수 카드 3칸 (내 점수 · nyam 점수 · 버블 점수 — 버블 확장 패널)
│   ├── 뱃지 행 (미슐랭, 블루리본, TV 등)
│   ├── L5: 내 기록 타임라인 (방문별 씬칩, 점수, 코멘트, 사진 썸네일)
│   ├── L6: 포지션 맵 (가격 × 분위기 사분면)
│   ├── L7: 실용 정보 (주소+미니맵, 영업시간, 전화, 메뉴)
│   ├── L8: 연결된 와인 (가로 스크롤 카드)
│   └── L9: 버블 멤버 기록 (버블별 필터 + 카드)
│
├── 와인 상세 (/wines/[id])
│   ├── L1: 히어로 (사진 캐러셀, 라벨 썸네일, 찜/공유)
│   ├── L2: 기본 정보 (이름, 생산자·빈티지, 타입칩+산지+품종)
│   ├── L3: 점수 카드 3칸 (식당과 동일 구조)
│   ├── 뱃지 행 (Grand Cru, Vivino, Wine Spectator 등)
│   ├── L5: 와인 맵 (산미 × 바디 사분면)
│   ├── L5b: 음식 페어링 태그 (WSET 7-카테고리)
│   ├── L6: 내 기록 타임라인 (씬칩 + 식당 연결 링크)
│   ├── L7: 와인 정보 (품종, 산지, 도수, 바디, 산미 등 2-컬럼 표)
│   ├── L8: 연결된 식당 (가로 스크롤 카드)
│   └── L9: 버블 멤버 기록
│
├── 기록 상세 (/records/[id])
│   ├── 헤더 (대상명 + 방문일 + 씬칩)
│   ├── 사분면 미니맵
│   ├── 만족도 점수 + 컬러바
│   ├── 아로마 팔레트 (와인만)
│   ├── 코멘트, 사진 갤러리, 메뉴/팁
│   ├── 실용 정보 (가격, 동행자, 방문일)
│   ├── 획득 XP
│   └── 액션 (수정 · 삭제 · 버블에 공유)
│
├── 프로필 (/profile)
│   ├── 프로필 헤더 (아바타, 닉네임, 소개, 팔로워/팔로잉 수)
│   ├── 맛 정체성 카드 (Wrapped 공유 가능)
│   ├── 활동 요약
│   │   ├── 총 레벨/XP 카드
│   │   ├── 식당/와인 개별 레벨 게이지
│   │   ├── 이번 달 XP
│   │   └── 활동 히트맵 (월별 기록 밀도)
│   ├── 식당 통계 (맛 지도, 장르 차트, 점수 분포, 씬 차트)
│   ├── 와인 통계 (산지 맵, 품종 차트, 점수 분포, 아로마 프로필)
│   └── 최근 기록 타임라인
│
├── 버블러 프로필 (/users/[id])
│   ├── 상대방 프로필 정보 + 팔로우 버튼
│   ├── 취향 매칭도 (±5% 비교)
│   └── 공개 기록/통계 (프라이버시 설정에 따름)
│
├── 버블 (/bubbles)
│   ├── 버블 목록 (내 버블 + 탐색)
│   ├── 버블 상세 (/bubbles/[id])
│   │   ├── 히어로 (아이콘, 이름, 멤버 수, 가입/초대)
│   │   ├── 탭: 피드 | 랭킹 | 멤버
│   │   ├── 피드 탭 — 뷰 모드: 피드 / 카드 / 리스트
│   │   │   ├── 노션 스타일 필터 + 소팅
│   │   │   └── 리액션(want/check/fire) + 댓글
│   │   ├── 랭킹 탭 — 주간 Top N, 순위 변동
│   │   └── 멤버 탭 — 멤버 목록, 역할, 초대
│   └── 버블 생성 (이름, 아이콘, 설명, 가입 조건, 공개 범위)
│
├── 설정 (/settings)
│   ├── 계정 (닉네임, 소개, 아바타, 소셜 연동, 로그아웃, 탈퇴)
│   ├── 프라이버시 (프로필 공개 범위, 공개 항목, 버블 기본값, 버블별 커스텀)
│   ├── 알림 (종류별 on/off — 레벨업, 버블 요청, 팔로우, 방해 금지)
│   ├── 데이터 (내보내기, 캐시 삭제, 계정 삭제)
│   └── 정보 (이용약관, 개인정보처리방침, 오픈소스, 버전)
│
└── 알림 (인라인 드롭다운 — 별도 페이지 아님)
    ├── 레벨업 알림
    ├── 버블 가입 요청 (수락/거절)
    ├── 팔로우 요청 (수락/거절)
    ├── 버블 가입 승인
    ├── 팔로우 수락
    └── 하단: 알림 설정 → 링크
```

## 2. 핵심 사용자 플로우

### Flow A: 식당 기록하기 (핵심 루프)
```
FAB (+) → 근처 식당 / 검색 → 대상 선택
  → 만족도(1-100 게이지) → 사분면(가격×분위기) → 씬태그(6종) → 동행자
  → [저장] (Phase 1 완료)
  → (선택) 사진, 한줄평, 메뉴태그, 가격대, 재방문 의향 (Phase 2)
```

### Flow B: 와인 기록하기
```
FAB (+, 와인 탭) → 라벨 촬영(OCR+AI) / 검색 → 대상 선택
  → 만족도 → 사분면(산미×바디) → 아로마 팔레트(12섹터) → 씬태그(6종)
  → [저장] (Phase 1 완료)
  → (선택) 사진(3모드), 한줄평, 음식 페어링, 가격, 구매처, 빈티지 (Phase 2)
```

### Flow C: 재방문/재음용 기록
```
상세 페이지 FAB (+) → 사분면 (이전 위치 반투명 표시) → 씬태그 → 저장
```

### Flow D: 추천/팔로잉 → 탐색
```
홈 식당 탭 → 추천 서브탭 → 추천 카드 (AI/버블 기반) → 상세 페이지 → 기록 or 찜
홈 식당 탭 → 팔로잉 서브탭 → 팔로잉 유저의 기록 → 상세 페이지
```

### Flow E: 넛지 → 기록 완성
```
앱 진입 → 넛지 스트립 (미평가/식사시간/사진 감지) → "기록하기" → 기록 플로우 진입
```

### Flow F: 버블 공유 플로우
```
기록 상세 → "버블에 공유" → 버블 선택 → 공유 완료
버블 상세 → FAB (+) → 기록 플로우 → 저장 시 자동 공유
```

### Flow G: 온보딩 플로우
```
로그인 (카카오/구글/애플) → 인트로 화면 → 경험치 설명 → 버블 생성 → 버블 탐색 → 홈
```

## 3. 네비게이션 구조

### 3-1. 글로벌 헤더 (모든 페이지 공통)

```
[← 컨텍스트 백] nyam    bubbles 🔔 [아바타]
```

| 요소 | 기능 |
|------|------|
| ← 컨텍스트 백 | referrer 기반 라벨 (홈/프로필/버블 등) — 상세 페이지에서만 표시 |
| nyam | 로고, 홈 링크 |
| bubbles | 텍스트 버튼, `/bubbles` 이동 (--brand 컬러) |
| 🔔 | 알림 드롭다운 토글 (읽지 않은 알림 시 뱃지) |
| 아바타 [J] | 드롭다운 메뉴 (프로필, 설정) |

### 3-2. 홈 콘텐츠 탭 바

```
[식당 | 와인]  [뷰 순환] [지도] [통계]  [필터] [소팅]
```

| 요소 | 기능 |
|------|------|
| 식당 / 와인 | 도메인 전환 (밑줄 색상: 식당=--primary, 와인=--wine) |
| 뷰 순환 | 상세(카드) → 리스트(컴팩트) → 캘린더 순환 |
| 지도 | 방문 식당/와인 지도 뷰 토글 |
| 통계 | 통계 패널 토글 (접이식) |
| 필터 | 노션 스타일 다중 조건 필터 드로어 |
| 소팅 | 드롭다운 (최신순, 점수순, 방문순, 가격순 등) |

### 3-3. FAB (Floating Action Button)

| FAB | 위치 | 기능 |
|-----|------|------|
| FAB (+) 추가 | 우하단 | 기록 추가 바텀시트 → 식당/와인 선택 → 기록 플로우 |
| FAB (←) 뒤로 | 좌하단 | 이전 페이지 (상세 페이지에서만) |

### 3-4. 페이지 간 이동 맵

```
홈 (/)
├── 카드 탭 → 식당 상세 (/restaurants/[id])
├── 카드 탭 → 와인 상세 (/wines/[id])
├── FAB (+) → 기록 플로우 (모달)
├── bubbles → 버블 목록 (/bubbles)
├── 🔔 → 알림 드롭다운
├── 아바타 → 프로필 (/profile)
└── 아바타 → 설정 (/settings)

식당 상세 (/restaurants/[id])
├── ← → 홈 / 프로필 / 버블 (referrer 기반)
├── 기록 카드 → 기록 상세 (/records/[id])
├── 와인 카드 → 와인 상세 (/wines/[id])
└── FAB (+) → 기록 플로우 (식당 프리셋)

와인 상세 (/wines/[id])
├── ← → 홈 / 프로필 / 버블 (referrer 기반)
├── 기록 카드 → 기록 상세 (/records/[id])
├── 식당 카드 → 식당 상세 (/restaurants/[id])
└── FAB (+) → 기록 플로우 (와인 프리셋)

기록 상세 (/records/[id])
├── ← → 이전 페이지
├── 수정 → 기록 플로우 (프리필)
├── 삭제 → 확인 모달 → 이전 페이지
└── 버블 공유 → 버블 선택 모달

프로필 (/profile)
├── 기록 → 기록 상세
├── ⚙ → 설정 (/settings)
└── 팔로워/팔로잉 → 버블러 프로필 (/users/[id])

버블 상세 (/bubbles/[id])
├── ← → 버블 목록 / 홈
├── 피드 카드 → 식당/와인 상세
├── 멤버 → 버블러 프로필 (/users/[id])
└── FAB (+) → 기록 플로우 (자동 공유)
```

## 5. 구현 순서 (스프린트 계획 — P1+P2 통합)

> P2 테이블은 S1에서 전부 생성. P2 기능은 S7~S8에서 구현.
> 온보딩은 모든 기능 완성 후 마지막에 연결.

```
S1 ─── S2 ─── S3 ─┬─ S4
 (기반)  (코어)  (검색) │ (상세P1)
                    │
                    S5 ─── S6 ─── S7 ─── S8 ─── S9
                   (홈+추천) (XP+프로필) (버블) (통합) (온보딩+마무리)
```

| Sprint | 문서 | 산출물 | 선행 | 비고 |
|--------|------|--------|------|------|
