# RATING_ENGINE — 사분면 평가 시스템

> affects: RECORD_FLOW, SEARCH_REGISTER, RESTAURANT_DETAIL, WINE_DETAIL, HOME, RECOMMENDATION, PROFILE

---

## 1. 핵심 개념

**하나의 평가 = (X%, Y%, 만족도, 상황태그)**
- 3초 안에 완료되는 4차원 데이터
- 사분면 위치(특성 2축) + 점 크기(만족도) + 상황 태그

---

## 2. 사분면 축 정의

### 식당
| 축 | 0% | 100% | 의미 |
|----|-----|------|------|
| X축: 가격 | 저렴 | 고가 | 체감 가격 (양/가치 종합) |
| Y축: 분위기 | 캐주얼 | 포멀 | 격식 수준 |
| 크기: 만족도 | 1 | 100 | 전체 만족도 (위치와 독립) |

```
             포멀
              ↑
합리적 격식집  |  프리미엄 다이닝
              |
저렴 ─────────┼───────── 고가
              |
동네 맛집     |  트렌디 핫플
              ↓
             캐주얼
```

### 와인
| 축 | 0% | 100% | 의미 |
|----|-----|------|------|
| X축: 산미 | 낮음 | 높음 | 신맛의 강도 |
| Y축: 바디 | 가벼움 | 무거움 | 입 안의 무게감 |
| 크기: 만족도 | 1 | 100 | 전체 만족도 |
| **색상: 향** | — | — | 향 팔레트 중심 색상 (와인만) |

---

## 3. 조작 플로우

| 단계 | 행동 | 제스처 | 시간 |
|------|------|--------|------|
| 1. 탭 | 사분면에서 대략 위치 선택 | 1-finger tap | ~0.5초 |
| 2. 드래그 | 정밀 위치 조정 | 1-finger drag | ~1초 |
| 3. 릴리즈 | 위치 확정 → 만족도 모드 전환 | 손 떼기 | 자동 |
| 4. 상하 드래그 | 만족도(크기) 조절 | 점을 상하로 드래그 | ~1초 |
| 5. 태그 | 상황 선택 | 1-finger tap | ~0.5초 |
| 6. 완료 | 기록 버튼 | 1-finger tap | ~0.3초 |

### 제스처 충돌 방지
- **위치 모드**: 상하=Y축, 좌우=X축
- **크기 모드**: 상하=만족도, 좌우=비활성
- 모드 전환: 릴리즈 후 점 다시 터치 → 크기 모드
- 점 바깥 탭 → 위치 모드 복귀

---

## 4. 점(Dot) 비주얼 스펙

| 속성 | 위치 모드 | 크기 모드 | 확정 후 |
|------|----------|----------|---------|
| 테두리 | 2px solid white | 2px dashed accent | 없음 (그림자만) |
| 그림자 | 8px blur 40% | 12px blur 60% | 6px blur 30% |
| 색상 | accent (반투명) | 만족도 기반 변화 | 최종 색상 고정 |
| 표시 | X%, Y% (점 옆) | 만족도 점수 (점 위) | 만족도 (점 안/옆) |

---

## 5. 애니메이션 & 햅틱

| 이벤트 | 애니메이션 | 햅틱 |
|--------|----------|------|
| 사분면 탭 | ripple → 해당 사분면 확대 (0.3s ease-out) | light tap |
| 위치 드래그 | 점 따라감, 그림자 확대, X%/Y% 실시간 | 50% 중심선 지날 때 딸깍 |
| 위치 확정 | 바운스 후 안착, 테두리 실선→점선 | medium tap |
| 크기 드래그 | 점 커짐/작아짐, 숫자 변화, 색상 변화 | 10점 단위 딸깍 |
| 태그 선택 | 커지며 강조색 전환 (0.2s) | light tap |
| 기록 완료 | 점 "박히는" 느낌 + 체크마크 (0.5s→fade) | success |
| 90+ 도달 | 글로우 이펙트 | heavy tap |

## 6. 배경 레퍼런스

사분면 배경에 반투명 점으로 대표 식당/와인 표시:

**데이터 소스 우선순위**:
1. 같은 지역 사용자 평균 좌표 (데이터 충분 시)
2. 미슐랭/블루리본 사전 배치 (콜드스타트)
3. 자신의 과거 기록

**표시 규칙**:
- 최대 8~12개
- 투명도: 15~25%
- 내 기록: accent 계열, 타인: 회색
- 확대 시에만 이름 표시 (가까운 3~4개)

## 7. 와인 심화 평가

식당은 Step 1~6으로 끝. 와인은 추가 단계:

| Step | 내용 | 필수 |
|------|------|------|
| 1 | 사분면 (산미×바디) | 필수 |
| 2 | 향 팔레트 (영역 그리기) | 필수 |
| 3 | 복합성 (1차/2차/3차 토글) | 선택 |
| 4 | 여운 & 균형 (슬라이더) | 선택 |
| 5 | 만족도 확정 (자동산출 + 조정) | 필수 |
| 6 | 상황 태그 | 필수 |

### 아로마 팔레트 상세 스펙

**레이아웃**: 원형 (시계 방향 12시부터)
- 12섹터, 3개 링 (1차향/2차향/3차향)

**1차향 (과일/꽃) — 바깥 링**:
| 위치 | 향 | 색상 | hex |
|------|-----|------|-----|
| 12시 | 시트러스 (레몬/라임) | 밝은 노랑 | `#fde047` |
| 1시 | 사과/배 | 연두 | `#a3e635` |
| 2시 | 열대과일 (망고/파인) | 주황 | `#fb923c` |
| 3시 | 핵과 (복숭아/살구) | 연분홍 | `#fda4af` |
| 4시 | 붉은 베리 (딸기/라즈베리) | 빨강 | `#f87171` |
| 5시 | 검은 베리 (블랙베리/카시스) | 보라 | `#a855f7` |
| 6시 | 꽃 (장미/제비꽃) | 핑크 | `#f472b6` |
| 7시 | 흰 꽃 (아카시아/재스민) | 크림 | `#fef3c7` |

**2차향 (발효/숙성) — 중간 링**:
| 위치 | 향 | 색상 | hex |
|------|-----|------|-----|
| 8시 | 버터/브리오슈 | 베이지 | `#fde68a` |
| 9시 | 바닐라/토스트 | 갈색 | `#d97706` |
| 10시 | 향신료 (후추/정향) | 다크레드 | `#991b1b` |
| 11시 | 허브 (민트/타임) | 초록 | `#4ade80` |

**3차향 (숙성) — 안쪽 링**:
| 위치 | 향 | 색상 | hex |
|------|-----|------|-----|
| 중앙-좌 | 가죽/담배 | 다크브라운 | `#78350f` |
| 중앙-우 | 흙/미네랄 | 회색 | `#78716c` |
| 중앙-하 | 견과/초콜릿 | 밤색 | `#92400e` |

**인터랙션**:
- 터치: 섹터 탭 → 해당 영역 활성화 (50% opacity → 100%)
- 드래그: 인접 섹터로 드래그하면 연속 영역 칠하기
- 멀티터치: 비연속 영역도 선택 가능
- 셰이크: 마지막 선택 취소 (undo)
- 선택 영역의 가중 평균 색상 = 사분면 점 대표 색상 (`aroma_color`)

**자동 라벨**:
- 선택한 섹터의 향 이름을 자동으로 `aroma_labels[]`에 저장
- 예: `[시트러스, 열대과일, 바닐라]` → "시트러스, 열대과일, 바닐라 느낌의 와인"

**복합성 판단**:
- 1차향만 선택 → `complexity=1`
- 1차+2차 선택 → `complexity=2`
- 1차+2차+3차 선택 → `complexity=3`

**aroma_color 계산**:
- 선택된 모든 섹터 hex의 가중 평균 (활성화된 섹터 면적 비례)
- 결과 hex를 `records.aroma_color`에 저장

### 만족도 자동 산출 (Step 3,4 입력 시)
```
기본 50 + 복합성[0|+7|+15] + 여운%×0.1 + 균형%×0.15
→ clamp(1, 100) → 사용자 미세 조정
```

## 8. 확장 평가 (선택)

Quick(사분면+크기+태그) 이후 선택 입력:

### 식당
| 항목 | 입력 방식 |
|------|----------|
| 한줄평 | 텍스트 200자 |
| 사진 | 갤러리/카메라 복수 |
| 추천 메뉴 | 텍스트 태그 자동완성 |
| 사용팁 | 텍스트 |
| 함께 간 사람 | 사용자 태그/닉네임 |
| 가격 (1인) | 금액 입력/범위 |

### 와인
| 항목 | 입력 방식 |
|------|----------|
| 한줄평 | 텍스트 |
| 사진 | 라벨/잔 |
| 페어링 메모 | 텍스트 |
| 가격 | 금액 |
| 구매처/장소 | 검색 (식당 연결 가능) |
| 빈티지 | 연도 선택 |
