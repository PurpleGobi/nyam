## 완료 기준

```
□ 앱 헤더: nyam 로고(Comfortaa), bubbles(--brand), 알림 뱃지, 아바타 드롭다운
□ FAB: + 추가(우하단 44x44), ← 뒤로(상세 페이지에서)
□ 식당/와인 탭 전환 (밑줄 색상: 식당=--accent-food, 와인=--accent-wine)
□ 서브 네비: 식당(방문/찜/추천/팔로잉), 와인(시음/찜/셀러)
□ 상세→리스트→캘린더 뷰 순환 + 지도 뷰 토글
□ 노션 스타일 필터 (다중 조건 AND/OR) + 소팅 드롭다운
□ 통계 패널 토글 + 차트 렌더링
□ AI 인사 (5초 후 소멸, 기록 기반 개인화)
□ 넛지 스트립 (미평가/식사시간/사진 감지, 1개씩)
□ 추천 카드 표시 (기록 수 기반 점진 해금, 7종 혼합 정렬)
□ Discover: 지역 칩 + 검색 + 카드 리스트 (홈 내 서브스크린)
```
