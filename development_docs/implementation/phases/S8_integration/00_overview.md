## 완료 기준

```
□ 팔로우 → 이름+점수 맛보기 접근 (XP 불필요)
□ 맞팔로우 → 풀 액세스 (리뷰, 사진, 팁)
□ 식당/와인 상세 L9: 버블별 필터 + 멤버 기록 카드
□ 홈 팔로잉 서브탭: 팔로잉 유저의 기록 피드
□ 기록 상세 → "버블에 공유" → 버블 선택 → 완료
□ 버블러 프로필: 상대방 정보 + 취향 매칭도(±5%) + 팔로우 버튼
□ 프라이버시 설정 연동 (privacy_profile, visibility 토글 반영)
```
