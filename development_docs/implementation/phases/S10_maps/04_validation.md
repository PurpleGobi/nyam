# S10-T4: 검증

> 대상: S10-T1(지도 데이터), S10-T2(식당 지도), S10-T3(와인 산지 지도)

---

## 1. 기능 검증

### 식당 지도

| 테스트 | 동작 | 기대 결과 |
|--------|------|----------|
| Level 0 렌더링 | 프로필 식당 탭 | 세계 지도 + 방문 도시 마커 표시 |
| 마커 크기 | count 1/3/6 | 5px/7px/10px |
| 마커 탭 | 국가 마커 탭 | Level 1 도시 목록 전환 |
| 뒤로 | ← 버튼 | Level 0 복귀 |
| 빈 상태 | 방문 기록 0 | 안내 메시지 |

### 와인 산지 지도

| 테스트 | 동작 | 기대 결과 |
|--------|------|----------|
| Level 0 렌더링 | 프로필 와인 탭 | 세계 지도 + 국가 마커 + 타입 도트 |
| Level 0→1 | 국가 탭 | 국가 윤곽선 + 산지 fill |
| Level 1→2 | 산지 탭 | 산지 확대 + 서브 AOC |
| 미탐험 산지 | Level 1 | 점선 border, opacity 0.3 |
| 뒤로 (Level 2→1) | ← 버튼 | 국가 뷰 복귀 |
| 뒤로 (Level 1→0) | ← 버튼 | 세계 뷰 복귀 |
| 와인 타입 도트 | 각 타입 | 레드=#722F37, 화이트=#D4C98A, 로제=#E8A0B0, 스파클링=#C8D8A0 |
| 빈 상태 | 와인 기록 0 | 안내 메시지 |
