## 배경

S6-T2에서 프로필 지도를 구현할 때, 두 지도 모두 **SVG 국가/산지 윤곽선 좌표 데이터**와 **다단계 드릴다운 상태 관리**가 필요하여 간이 버전만 먼저 구현했다. S10에서 제대로 된 드릴다운 지도로 고도화한다.

### 현재 간이 버전

- **식당 지도** (`profile/restaurant-map.tsx`): 타원 대륙 + lat/lng 마커. 드릴다운 없음.
- **와인 산지 지도** (`profile/wine-region-map-simple.tsx`): 국가별 바 리스트 + 타입 도트. 드릴다운 없음.

### S10 목표

- **식당 지도**: 실제 대륙/국가 SVG path → 국가 탭 시 도시 목록 드릴다운
- **와인 산지 지도**: 세계 → 국가(산지 fill) → 산지(서브 AOC) 3단계 드릴다운
