## 2. 레벨 표시

- 메인 3개: 가장 높은 레벨 3개 강조 표시
- 각 뱃지: `[동네명] Lv.N` + 레벨 색상 (green/blue/purple/orange/gold)
- "전체 레벨 보기" → 모든 지역/와인 레벨 목록 (접이식)

## 4. 사분면 차트

- 전체 기록 분포를 하나의 사분면에 표시
- **식당 기록**: 점 fill = 만족도 색상, 점 테두리 = 상황 태그 색상
- **와인 기록**: 점 fill = 향 팔레트 색상, 점 테두리 = 상황 태그 색상
- 점 크기 = 만족도 (1~100 → 12~54px)
- 밀도 높은 영역 강조
- Phase 3: 자동 생성 한줄 설명
