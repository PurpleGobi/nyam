## 2. 나의 미식 정체성

- 기록 데이터 기반 자동 생성 텍스트 + 태그
- 주력 동네 + 선호 분위기 + 와인 취향 요약
- **공유 버튼** → Wrapped 공유 화면으로 이동
- Phase 1: 규칙 기반 템플릿, Phase 3: AI 생성

### 게이지 조합 매트릭스

| | 개인정보: 최소 | 개인정보: 보통 | 개인정보: 공개 |
|---|---|---|---|
| **디테일: 심플** | 익명 + 태그만 | 닉네임 + 숫자 + 태그 | + 버블 표시 |
| **디테일: 보통** | 익명 + 태그 + 레벨 | 닉네임 + 숫자 + 태그 + 레벨 | + 버블 표시 |
| **디테일: 상세** | 익명 + 태그 + 레벨 + 최애 | 닉네임 + 숫자 + 태그 + 레벨 + 최애 | 전체 공개 |

### 카테고리 필터

- **전체**: 식당 + 와인 통합 정보
- **식당**: 지역/장르 레벨 + 식당 최애 + 식당 기록 수만 표시
- **와인**: 산지/품종 레벨 + 와인 최애 + 와인 기록 수만 표시

### 저장/공유

- 카드를 이미지로 렌더링 → 저장 또는 OS 공유 시트
- 반응형 레이아웃: 게이지에 따라 카드 높이가 자동 조절

## 4. 경험치 & 레벨

- **식당 / 와인** 탭 필터로 구분 (디자인 시스템 `.tabs` 패턴)
- 각 탭 안에서 **두 컬럼** 으로 경험치 축 분류:
  - **식당 탭**: 지역(을지로, 광화문…) | 장르(일식, 한식…)
  - **와인 탭**: 산지(보르도, 부르고뉴…) | 품종(카베르네 소비뇽, 피노 누아…)
- 각 축: 레벨 + XP 진행 바
- 레벨 색상: green(1~3) → blue(4~5) → purple(6~7) → orange(8~9) → gold(10)

---

## 5. 최근 경험치

- 최근 XP 획득 내역 4~5개 표시
- 아이콘으로 구분: 새 장소 / 와인 / 재방문 / 마일스톤
- 각 항목: 장소명 + 축 + XP 금액
