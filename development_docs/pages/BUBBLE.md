# BUBBLE — 버블 & 소셜 (Phase 2)

> depends_on: DATA_MODEL, AUTH, XP_SYSTEM, DESIGN_SYSTEM
> route: /bubbles, /bubbles/[id]

---

## 1. 핵심 원칙

- **기록이 먼저, 소셜은 그 다음** — 소셜 없이도 유용
- **폐쇄가 신뢰를 만든다** — 광고/가짜 리뷰 구조적 불가
- **공유는 선택적** — 원하는 기록만 특정 버블에
- **피드는 시간순** — 알고리즘 피드 없음 (신뢰 보호)

---

## 2. 버블 유형

| 유형 | 구조 | 사용 예시 |
|------|------|----------|
| 양방향 | 모든 멤버 공유+반응 | 직장 맛집, 와인 동호회 |
| 열람 전용 | 운영자만 공유 | 맛잘알 큐레이션, 소믈리에 |

---

## 3. 버블 생성

```
1. [+] 버블 만들기
2. 이름 (20자) + 설명 (100자, 선택) + 유형 + 아이콘
3. 초대 설정: 링크 (만료기간) or 직접 초대. 승인 필요 ON/OFF
4. 완료 → 버블 상세
```

---

## 4. 공유 플로우

```
방법 1: 기록 직후 → "버블에 공유하기" → 버블 선택 → 공유
방법 2: 기존 기록 → 공유 아이콘 → 버블 선택
방법 3: 필터 일괄 → "광화문 + 80점 이상" → 5개 한번에 공유
```

**공유 시 포함**: 식당/와인명, 사분면 좌표, 만족도, 상황, 한줄평, 사진, 메뉴, 팁, 날짜
**공유자 선택**: 가격, 함께 간 사람 (민감 정보 ON/OFF)
**공유 취소**: 언제든 철회 가능 (달린 댓글은 고아 상태로 유지)

---

## 5. 리액션

| 유형 | 대상 | 의미 |
|------|------|------|
| 좋아요 | 기록, 댓글 | 공감/도움 |
| 찜 | 기록의 식당/와인 | Wishlist에 추가 |

2개만. 이모지 리액션은 오버킬.

---

## 6. 댓글

- 최대 300자, 텍스트만
- 익명 옵션 (버블 설정에서 허용/금지)
- 본인만 삭제 가능

---

## 7. 버블 상세 페이지

```
┌──────────────────────────────────────┐
│ [←] 직장 맛집                         │
│ 멤버 8명 · 기록 47개                    │
│ [초대하기] [설정]                       │
│                                       │
│ [피드] [식당] [와인] [통계]             │  탭
│                                       │
│ ── 피드 탭 (기본) ──                   │
│ 최신 공유 기록 (시간순)                  │
│ 각 카드: 공유자+만족도+상황+한줄평       │
│        + ♡수 + 💬수                    │
│                                       │
│ ── 식당/와인 탭 ──                     │
│ 공유된 모든 식당/와인 목록               │
│ 정렬: 최신/만족도/기록수/찜수            │
│ 필터: 지역, 장르, 상황, 만족도           │
│                                       │
│ ── 통계 탭 ──                         │
│ 공유 랭킹 · 인기 식당 Top 5            │
│ 장르 분포 · 사분면 분포                 │
└──────────────────────────────────────┘
```

## 8. 홈화면 버블 피드

```
── 버블 ──
[아바타] 김영수 · 직장 맛집
미진 — 93점 · 혼밥
"메밀국수 진짜 맛있다"
♡ 4  💬 2  🔖

[아바타] 박소연 · 와인 모임
Penfolds Grange — 96점
♡ 8  💬 5  🔖

[모든 버블 보기 →]
```

- 정렬: 시간순 (알고리즘 없음)
- 필터: 전체/특정 버블, 식당/와인, 상황 태그

---

## 9. 식당/와인 상세 — 소셜 레이어

상세페이지 Layer 9에 버블 기록 표시:
- 내 기록 아래에 배치
- 버블 필터 가능 (복수 소속 시)
- 각 기록: 아바타 + 닉네임 + 버블명 + 만족도 + 사분면좌표 + 한줄평
- 최대 5개 + "더보기"
- 사분면 차트에 버블 멤버 점도 반투명(30%)으로 표시

---

## 10. 경험치 & 소셜 신뢰

- 기록 옆에 **관련 레벨만** 표시 (을지로 기록 → 을지로 Lv.7)
- "경험자 순" 정렬 = 필터/정렬 옵션. 자동 가중치 아님

---

## 11. 알림

| 유형 | 트리거 |
|------|--------|
| 새 공유 | 내 버블에 새 기록 |
| 리액션 | 내 기록에 좋아요 |
| 댓글 | 내 기록에 댓글 |
| 찜 | 내 기록의 식당/와인 찜 |
| 초대 | 버블 초대 |
| 리뷰 | 같은 식당/와인에 다른 멤버 기록 |

설정: 버블별 ON/OFF, 방해 금지 23:00~08:00, 미읽음 = 도트
