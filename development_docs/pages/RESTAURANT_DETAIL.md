## 2. 섹션별 상세

### Layer 1: 히어로 (사진 캐러셀)
- 높이: h-56 (224px)
- 사진 없을 때: `bg-neutral-100` + 장르 아이콘
- 좌우 스와이프, dot indicator

### Layer 2: 아이덴티티
- 식당명: H2 (1.563rem, semibold, neutral-800)
- 장르 · 지역: Small (0.875rem, neutral-500)
- 가격대: ₩~₩₩₩₩

### Layer 3: 평가 요약
- **Nyam 점수** (좌): Display 2rem bold, primary-500. "N명이 기록" caption
- **외부 평점** (우): N/K/G 알파벳 약자 + 색상 구분 (N=초록, K=노랑, G=파랑)
- **권위 뱃지**: `rounded-lg bg-amber-50 text-amber-700 px-2 py-0.5`
- 기록 0개: 이 영역 숨김

### Layer 4: 액션 바
- [기록 남기기]: Primary (Coral Orange, flex-1). 기록 있으면 "재방문 기록"
- [♡]: Wishlist 토글
- [공유]: Share2 아이콘
- **스크롤 시**: 액션 바 지나면 "기록 남기기" 하단 고정 (floating)

### Layer 5: 나의 기록
**기록 있음**: 방문 N회 · 최근 날짜 → 타임라인 (최신순)
- 각 카드: `● 날짜 — 상황칩 | 점수 · 한줄평 | [사진]`
- 탭 → 기록 상세

**기록 없음**: "아직 방문 기록이 없어요" + [첫 기록 남기기] CTA

**찜만 한 상태**: "찜한 식당이에요. 방문하셨나요?" + [기록 남기기]

### Layer 6: 사분면 분포
- 기록 2개 이상일 때만 표시
- 모든 기록을 2D 좌표 위에 점으로 표시
- **점 fill 색상** = 만족도 기반 (DESIGN_SYSTEM 만족도 색상 매핑: 1~30 레드 → 86~100 블루)
- **점 테두리 색상** = 상황 태그 색상
- **점 크기** = 만족도 (1~100 → 12~54px)
- 내 기록: 불투명, 타인/버블(P2): 반투명(30%)
- 인기 메뉴: 태그 빈도 순 표시

### Layer 7: 실용 정보
- 주소 + 미니 지도 (h-32, 탭→외부 지도 앱)
- 영업 상태: `text-green-600` (영업중) / `text-neutral-400` (종료)
- 전화: `tel:` 링크
- 메뉴판: 접이식

### Layer 8: 식당-와인 연결
- 이 식당에서 와인 기록 있을 때만
- 가로 스크롤 카드: 라벨 썸네일 + 와인명 + 점수

### Layer 9: 버블 기록 (Phase 2)
- Phase 1: 숨김
- Phase 2: 버블별 기록, 좋아요/댓글

## 4. 인터랙션

- 스크롤 < 히어로: 투명 헤더
- 스크롤 >= 히어로: 고정 헤더 + 식당명 (backdrop-blur)
- 기록 CTA: 스크롤 시 하단 floating
- 접이식: 200ms ease-in-out
