# SETTINGS — 설정

> depends_on: AUTH, DATA_MODEL, DESIGN_SYSTEM

---

## 1. 설정 항목

### 계정
- 닉네임 변경
- 한줄 소개 변경
- 아바타 변경 (갤러리/카메라)
- 로그아웃
- 계정 삭제 (확인 모달 필수)

### 프라이버시
| 항목 | 옵션 | 기본값 |
|------|------|--------|
| 프로필 공개 | 전체 / 버블 멤버만 / 비공개 | 버블 멤버만 |
| 기록 공개 | 공유한 것만 / 전체 / 비공개 | 공유한 것만 |
| 소속 버블 표시 | ON / OFF | ON |
| 레벨 표시 | ON / OFF | ON |
| 사분면 표시 | ON / OFF | ON |

### 알림
| 항목 | 옵션 | 기본값 |
|------|------|--------|
| 넛지 (유형별) | ON / OFF | ON |
| 푸시 알림 | ON / OFF | ON |
| 방해 금지 | 시간 설정 | 23:00~08:00 |
| 버블 알림 (버블별) | ON / OFF | ON |

### 데이터
- 데이터 내보내기 (JSON/CSV)
- 캐시 삭제
- 위치 권한 관리 (OS 설정으로 이동)
- 갤러리 접근 권한 관리

### 정보
- 이용약관
- 개인정보처리방침
- 오픈소스 라이선스
- 앱 버전

---

## 2. 계정 삭제

```
[계정 삭제] 탭
→ 확인 모달: "정말 삭제하시겠습니까? 모든 기록이 삭제됩니다."
→ [취소] [삭제]
→ 삭제 시:
  - users soft delete
  - records 유지 (anonymous 처리) or 완전 삭제 (사용자 선택)
  - 버블 멤버 탈퇴 처리
  - 소셜 로그인 연결 해제
```
