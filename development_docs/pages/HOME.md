## 2. AI 멘트 (헤더 아래)

- 헤더 바로 아래, 넛지 카드 위에 위치
- **5초 후 자동 소멸** (opacity + max-height 트랜지션으로 fade out)
- 한 번 사라지면 다시 나타나지 않음 (세션 내)
- **탭하면 멘트에 언급된 식당 상세 페이지로 이동** (식당 언급이 없는 멘트는 탭 무반응)

### 시간대별 멘트 풀

| 시간대 | 예시 | 연결 식당 |
|--------|------|-----------|
| 아침 (6~11시) | "어제 스시코우지는 어떠셨어요?" | 스시코우지 |
| 점심 (11~15시) | "지난번 미진에서 88점 주셨잖아요" | 미진 |
| 저녁 (15~21시) | "데이트라면 스시코우지 다시 가셔도" | 스시코우지 |
| 밤 (21~6시) | "이번 주 기록 3건 — 꾸준히 잘 하고 계세요" | 없음 |

- 사용자 기록 기반으로 멘트 생성 (기록한 식당명, 점수, 방문 빈도 반영)
- 서브텍스트: `● nyam AI · 나의 기록 기반`

## 3. 내 기록 타임라인

- 최신순, 무한 스크롤
- 각 기록 카드:
  ```
  ● 날짜 — 상황 태그 칩
    식당/와인명 · 만족도 (Bold, primary)
    한줄평 (있으면, Small, neutral-600)
    [사진 썸네일 가로 스크롤] (있으면)
  ```
- **미평가 기록** (status='checked'):
  ```
  ● (미평가) 식당명
    장르 · 지역  [평가하기 →]
  ```
- 기록 탭 → 기록 상세 or 식당/와인 상세
- 미평가 "평가하기" 탭 → 사분면 평가 바로 진입

## 4. "오늘 뭐 먹지?" (접이식 추천)

### 접힌 상태
```
┌──────────────────────────────┐
│ 📍 오늘 뭐 먹지?    탭하여 추천 │
└──────────────────────────────┘
```

### 펼친 상태
```
┌──────────────────────────────┐
│ 상황 필터:                    │
│ [전체] [혼밥] [데이트] [회식]  │
│                              │
│ 도쿄등심         92점 · 회식  │
│ 여문식당         91점 · 친구  │
│ 미진            88점 · 혼밥  │
│                              │
│ 3개월 전 찜:                 │
│ 스시사이토 (강남) — 가볼까요?  │
└──────────────────────────────┘
```

기록 < 5개: 권위 추천 (미슐랭/블루리본/외부 평점)으로 대체

## 6. 넛지 카드

앱 진입 시 최우선순위 1개만:

### 사진 감지
```
┌──────────────────────────────┐
│ 📷 최근 사진에서 발견          │
│ [사진 썸네일] 3/19 12:34      │
│ 이 사진으로 기록할까요?        │
│ [기록하기] [무시]              │
└──────────────────────────────┘
```

### 미평가 기록
```
┌──────────────────────────────┐
│ 아직 평가하지 않은 곳이 3곳    │
│ 미진 — [평가하기]             │
│ [나중에]                      │
└──────────────────────────────┘
```

## 8. 상태별 홈 화면

| 상태 | 화면 |
|------|------|
| 기록 0개 (온보딩 직후) | 미평가 목록 + "첫 기록을 남겨보세요" CTA |
| 기록 1~4개 | 타임라인 + 넛지 카드 + 권위 추천 |
| 기록 5~9개 | 타임라인 + 재방문 추천 + 넛지 |
| 기록 10~19개 | 타임라인 + 상황별 추천 (상황 2종 이상, 각 3개+ 기록) |
| 기록 20+ (와인 기록 3개+) | 타임라인 + 상황별 추천 + 와인 추천 |
