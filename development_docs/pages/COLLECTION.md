## 1. 개요

홈 탭의 최신 미리보기와 별도로, 내 식당/와인 기록 전체를 브라우징하는 페이지.
컴팩트 리스트 + 노션 스타일 필터/소팅으로 대량 데이터를 효율적으로 탐색.

### 진입 경로
| 경로 | 동작 |
|------|------|
| 홈 와인 탭 → 우측 목록 아이콘 | 와인 탭 활성 상태로 진입 |
| 홈 식당 탭 → "전체 보기" | 식당 탭 활성 상태로 진입 |
| 프로필 → 내 기록 전체 | 마지막 사용 탭으로 진입 |
| 와인 대시보드 수치 탭 | 해당 필터 프리셋 적용된 채 진입 |

### 홈 목록과의 차이

| | 홈 탭 | 컬렉션 |
|--|-------|--------|
| 데이터 | 필터별 최신 N개 미리보기 | 전체 기록 |
| 카드 | L/R 플레이스 카드 (3소스) | 컴팩트 한 줄 리스트 |
| 필터 | 상황 칩 단일 선택 | 노션 스타일 다중 조건 |
| 소팅 | 순환형 (최신→점수→가나다) | 드롭다운 5종 |

## 3. 노션 스타일 필터 시스템

### 3-1. 동작 원리

**노션 DB 필터 로직을 따른다:**

- **필터 버튼** 클릭 → 속성 선택 드롭다운
- 속성 선택 → `Where [속성] is [값]` 규칙이 인라인 추가
- 2번째 규칙부터 `And` / `Or` 접속사 표시
- **접속사 클릭** → And ↔ Or 토글 (노션과 동일: 모든 규칙이 같은 접속사 공유)
- **속성 클릭** → 다른 속성으로 순환
- **값 클릭** → 해당 속성의 다른 옵션으로 순환
- **✕** → 해당 규칙 제거
- **+ 필터 추가** → 새 규칙 추가

### 3-2. 접속사 규칙 (노션 동일)

```
규칙 1개:     Where [속성] is [값]
규칙 2개+:    Where [속성] is [값]
              And   [속성] is [값]     ← And 클릭 시 전체가 Or로 전환
              And   [속성] is [값]

Or 전환 후:   Where [속성] is [값]
              Or    [속성] is [값]     ← Or 클릭 시 전체가 And로 전환
              Or    [속성] is [값]
```

- **And**: 모든 조건을 동시에 만족하는 항목만 표시
- **Or**: 조건 중 하나라도 만족하면 표시
- 한 필터 그룹 내에서 And/Or 혼합 불가 (노션과 동일)

### 3-3. 필터 속성

#### 식당 필터
| 속성 | 값 옵션 |
|------|---------|
| 상태 | 방문 / 찜 / 추천 |
| 상황 | 혼밥 / 데이트 / 친구 / 가족 / 회식 / 술자리 |
| 장르 | 한식 / 일식 / 양식 / 중식 / 이탈리안 / 프렌치 |
| 지역 | 강남 / 을지로 / 청담 / 이태원 / 홍대 / 광화문 |
| 점수 | 90+ / 80~89 / 70~79 / ~69 |

#### 와인 필터
| 속성 | 값 옵션 |
|------|---------|
| 상태 | 시음 / 찜 / 셀러 |
| 상황 | 혼술 / 데이트 / 모임 / 페어링 / 선물 / 시음회 |
| 타입 | Red / White / Rosé / Sparkling / Dessert |
| 산지 | France / Italy / USA / Chile / Australia / NZ |
| 점수 | 90+ / 80~89 / 70~79 / ~69 |

### 3-4. 필터 프리셋 진입

와인 대시보드에서 수치를 탭하면 해당 필터가 적용된 채 컬렉션으로 이동:

| 대시보드 탭 위치 | 적용 필터 |
|-----------------|----------|
| 품종 차트 → "카베르네 소비뇽" 행 탭 | 타입 is Red + (품종 필터 추가 예정) |
| 점수 분포 → "90+" 구간 탭 | 점수 is 90+ |
| 월별 소비 → "3월" 막대 탭 | (날짜 필터 향후 추가) |
| 산지 지도 → "France" 영역 탭 | 산지 is France |

## 5. 컴팩트 리스트 아이템

### 식당 아이템

```
[48px thumb]  식당명                      95
              장르 · 지역               3일 전
              [상황] [소스태그]
```

| 요소 | 스펙 |
|------|------|
| 썸네일 | 48×48px, border-radius 10px, 그래디언트 배경 |
| 식당명 | 14px, weight 700, 1줄 말줄임 |
| 메타 | 12px, 장르 · 지역 |
| 태그 행 | 상황(회색), 소스(나: accent, 버블: wine, 웹: hint) |
| 점수 | 16px, weight 800, 왼쪽에 8px 컬러 도트 |
| 날짜 | 11px, hint 색상 |
| 미평가 | 회색 처리 + "평가하기" CTA |

### 와인 아이템

```
[36×52 bottle]  와인명                    94
                타입 · 산지             1주 전
                [품종] [상황]
```

| 요소 | 스펙 |
|------|------|
| 썸네일 | 36×52px 병 모양, border-radius 6px |
| 색상 | Red: 어두운 그래디언트, White: 밝은 골드, Rosé: 핑크 |

| 품종 태그 | wine-light 배경, wine 텍스트 |

## 7. 데이터 소스

```typescript
// 쿼리 구조
SELECT r.*, p.name, p.genre, p.area
FROM records r
JOIN places p ON r.target_id = p.id
WHERE r.user_id = :userId
  AND r.target_type = :tab  -- 'restaurant' | 'wine'
  AND [dynamic filter conditions]  -- 노션 필터 규칙 → SQL 변환
ORDER BY [sort option]
```

### 필터 → SQL 변환

```
And 모드: WHERE cond1 AND cond2 AND cond3
Or 모드:  WHERE cond1 OR cond2 OR cond3
```
