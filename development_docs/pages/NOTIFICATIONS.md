## 2. 섹션별 상세

### 헤더
- [←] 뒤로가기 → 홈
- "알림" (H2, semibold)
- [⚙️] 알림 설정 페이지로 이동

### 날짜 그룹
- **오늘**: 오늘 발생한 알림
- **이번 주**: 오늘 제외, 이번 주 내 알림
- **이전**: 이번 주 이전 알림
- 각 그룹 구분선: Small (0.875rem, neutral-400, semibold) + 구분선

### 알림 카드
```
┌──────────────────────────────────────┐
│ [아이콘]  제목 (Body, semibold)       │
│          본문 (Small, neutral-600)    │
│          타임스탬프 (XS, neutral-400) │
│                              [● 미읽음] │
└──────────────────────────────────────┘
```
- 미읽음: `bg-neutral-50` + 우측 `●` 도트 (primary 색상, 8px)
- 읽음: `bg-background` (투명), 도트 없음
- 카드 패딩: `px-4 py-3`
- 탭 → 관련 페이지로 이동 + 자동 읽음 처리

### 스와이프 액션
- 좌측 스와이프: "읽음 처리" (이미 읽음이면 "삭제")
- 스와이프 배경: `bg-neutral-200` + 아이콘
- 애니메이션: 200ms ease-out

## 3. 알림 유형

### Phase 1: 넛지 + 성장

| 유형 | 아이콘 | 제목 | 본문 | 탭 이동 |
|------|--------|------|------|---------|
| 사진 감지 | 📷 | 최근 사진에서 맛집을 발견했어요 | 기록해볼까요? | RECORD_FLOW (사진 첨부) |
| 미평가 | 📝 | 아직 평가하지 않은 곳이 있어요 | {식당명} 외 {N}곳 | 미평가 기록 → 사분면 진입 |
| 식사 후 | 🍽️ | 방금 식사하셨나요? | 기록을 남겨보세요 | RECORD_FLOW |
| 새 동네 | 📍 | 새로운 동네에서 기록을 남겼어요 | {area}의 첫 기록! | 기록 상세 |
| 주간 리마인드 | 📊 | 이번 주 기록 {N}개 | 지난주보다 {diff}개 {증감} | 홈 |
| 레벨업 | 🎖️ | {axis_value} 레벨 {level} 달성! | 기록이 쌓이고 있어요 | 프로필 |

### Phase 2: 버블 소셜

| 유형 | 아이콘 | 제목 | 본문 | 탭 이동 |
|------|--------|------|------|---------|
| 버블 초대 | 🫧 | {nickname}님이 '{bubble_name}'에 초대했어요 | [수락] [거절] | 버블 상세 |
| 새 공유 | 📮 | {nickname}님이 기록을 공유했어요 | {restaurant/wine_name} · {satisfaction}점 | 버블 피드 → 해당 기록 |
| 좋아요 | ♡ | {nickname}님이 좋아요를 눌렀어요 | {restaurant/wine_name} 기록 | 기록 상세 |
| 댓글 | 💬 | {nickname}님이 댓글을 남겼어요 | "{comment_preview}" | 기록 상세 → 댓글 |
| 찜 | 🔖 | {nickname}님이 내 기록의 {target}을 찜했어요 | — | 기록 상세 |
| 팔로우 | 👤 | {nickname}님이 나를 팔로우합니다 | — | 상대 프로필 |

## 5. HOME 연동: 미읽음 뱃지

- 홈 헤더 🔔 아이콘에 미읽음 개수 뱃지 표시
- 뱃지: `absolute -top-1 -right-1`, `bg-red-500 text-white rounded-full min-w-[18px] h-[18px]`
- 10개 이상: "9+"
- 0개: 뱃지 숨김
- 미읽음 수 = `notifications WHERE user_id = me AND is_read = false` COUNT

## 6. 알림 설정 (⚙️)

```
┌──────────────────────────────────────┐
│ [←] 알림 설정                         │
│                                      │
│ ── 넛지 알림 ─────────────────────    │
│ 사진 감지              [토글 ON]      │
│ 미평가 리마인드         [토글 ON]      │
│ 식사 후 알림           [토글 OFF]     │
│ 주간 리포트            [토글 ON]      │
│                                      │
│ ── 성장 알림 ─────────────────────    │
│ 레벨업                 [토글 ON]      │
│                                      │
│ ── 버블 알림 (P2) ────────────────    │
│ 새 공유                [토글 ON]      │
│ 좋아요                 [토글 ON]      │
│ 댓글                   [토글 ON]      │
│ 초대                   [토글 ON]      │
│ 찜                     [토글 OFF]     │
│                                      │
│ ── 방해 금지 ─────────────────────    │

```

- 각 토글 → notifications 테이블의 type 필터에 반영
- 방해 금지: 설정 시간 내 푸시 알림 차단 (인앱은 누적)

## 7. 인터랙션

- 무한 스크롤: 20개씩 로드, `created_at DESC`
- 카드 탭: 관련 페이지 이동 + `is_read = true` UPDATE
- 스와이프 좌: 읽음 처리 / 삭제
- 버블 초대 알림: 인라인 [수락] [거절] 버튼 (탭 이동 없이 처리)
- Pull-to-refresh: 최신 알림 갱신

## 8. 데이터 소스

| UI 요소 | 소스 | 갱신 |
|---------|------|------|
| 알림 목록 | notifications | 실시간 (Supabase realtime 구독) |

| 알림 설정 | user_notification_settings (별도 테이블 또는 users JSONB) | 사용자 변경 시 |
| 넛지 이력 | nudge_history | 넛지 발생 시 |

## 9. Phase 구분

### Phase 1
- 넛지 알림: 사진 감지, 미평가, 식사 후, 새 동네, 주간 리마인드
- 성장 알림: 레벨업
- 알림 설정 (넛지 + 성장 섹션)
- 홈 미읽음 뱃지

### Phase 2
- 버블 소셜 알림: 초대, 새 공유, 좋아요, 댓글, 찜, 팔로우
- 알림 설정 (버블 섹션 추가)
- 버블별 ON/OFF 세부 설정
