## 2. 드롭다운 UI 상세

### 트리거
- 홈 헤더 🔔 아이콘 탭 → 드롭다운 팝업 토글
- 외부 영역 탭 → 닫기
- 스크롤 시 → 닫기

### 드롭다운 컨테이너
```
position: absolute
top: calc(헤더 높이 + 8px)
right: -16px  (우측 여백 확보)
width: 300px
max-height: 400px
overflow-y: auto
background: var(--bg-elevated)
border: 1px solid var(--border)
border-radius: 14px
box-shadow: 0 6px 24px rgba(0,0,0,0.1)
z-index: 200
```

### 헤더 바
- "알림" (Body, semibold) 좌측
- "모두 읽음" (Small, accent-social) 우측 → 전체 읽음 처리

### 알림 아이템
```
┌──────────────────────────────────────┐
│ [아이콘]  제목 (13px)                 │
│          본문 (12px, text-sub)        │
│          타임스탬프 (11px, text-hint) │
│                              [● 미읽음] │
└──────────────────────────────────────┘
```
- 아이콘: 단색 (`text-sub`), 16px, 배경 없음
- 미읽음: 제목 `font-weight: 600` + 우측 `●` 도트 (brand 색상, 6px)
- 읽음: 제목 `font-weight: 500`, 도트 없음
- 패딩: `px-3.5 py-2.5`
- 탭 → 관련 페이지로 이동 + 자동 읽음 처리 + 드롭다운 닫기
- 아이템 사이 구분: 1px `border-bottom` (구분선만으로 영역 분리)

### 하단 링크
- "알림 설정 →" 텍스트 → `/settings` (알림 섹션으로 스크롤)
- 중앙 정렬, Small, accent-social, `py-3`

### 빈 상태
```
┌──────────────────────────────────────┐
│            🔔                        │
│     아직 알림이 없어요                │
└──────────────────────────────────────┘
```

## 3. 알림 유형 (5가지만)

| 유형 | 아이콘 | 제목 | 본문 | 탭 이동 | 인라인 액션 |
|------|--------|------|------|---------|------------|
| 버블 가입신청 | circle-dot | {nickname}님이 '{bubble_name}' 가입을 신청했어요 | — | 버블 상세 | [수락] [거절] |
| 버블 가입승인 | circle-check | '{bubble_name}' 가입이 승인되었어요 | — | 버블 상세 | — |
| 레벨업 | trophy | {axis_value} 레벨 {level} 달성! | 기록이 쌓이고 있어요 | 프로필 | — |
| 팔로우 요청 | user-plus | {nickname}님이 팔로우를 요청했어요 | — | 상대 프로필 | [수락] [거절] |
| 팔로우 수락 | user-check | {nickname}님이 팔로우를 수락했어요 | — | 상대 프로필 | — |

### 인라인 액션 버튼
- [수락]: `bg-accent-social text-white`, 12px semibold, `px-3 py-1.5 rounded-lg`
- [거절]: `bg-transparent text-sub border border-border`, 동일 사이즈
- 처리 후: 버튼 → 결과 텍스트 ("수락 완료" / "거절됨") 치환

## 4. HOME 연동: 미읽음 뱃지

- 홈 헤더 🔔 아이콘에 미읽음 뱃지 dot 표시
- 뱃지: `absolute -top-1 -right-1`, `bg-brand rounded-full w-[7px] h-[7px]`
- 미읽음 0개: 뱃지 숨김
- 미읽음 수 = `notifications WHERE user_id = me AND is_read = false` COUNT

---

## 5. 인터랙션

- 최대 20개 표시 (최신순, `created_at DESC`)
- 아이템 탭: 관련 페이지 이동 + `is_read = true` UPDATE + 드롭다운 닫기
- 버블 가입신청 / 팔로우 요청: 인라인 [수락] [거절] 버튼 (탭 이동 없이 처리)
- "모두 읽음" 탭: 전체 `is_read = true` UPDATE
- 드롭다운 닫기: 외부 탭, 스크롤, ESC, 벨 아이콘 재탭

## 7. 알림 설정 → SETTINGS 페이지로 편입

알림 설정은 `/settings` 페이지의 "알림" 섹션에서 관리:

| 항목 | 옵션 | 기본값 |
|------|------|--------|
| 푸시 알림 | ON / OFF | ON |
| 레벨업 알림 | ON / OFF | ON |
| 버블 가입 알림 | ON / OFF | ON |
| 팔로우 알림 | ON / OFF | ON |
| 방해 금지 | 시간 설정 | 23:00~08:00 |
